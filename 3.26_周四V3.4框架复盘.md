### 周四003 斯洛伐克 vs 科索沃（3-4 客胜）❌

**H2d模式：同向** ★★★

| 维度 | 数据 | 信号 |
|------|------|------|
| 澳门方向 | H+4.0% D-5.5% A0% | H升D降 |
| 竞彩方向 | H+5.6% D-0.7% A-8.2% | H升A降 |
| 澳门推荐 | 客胜 | |
| 近况差 | -3 | 客队优势 |
| 离散度 | 5.5→0.4% | 缩小（高度一致） |

**V3报告判断**：平局（规律五：主升>5%）
**实际结果**：3-4 客胜 ❌

**V3.4框架重新分析**：
- 离散度从5.5%缩小到0.4% → 市场共识极强
- 澳门H升4.0% → 推离主队
- 澳门D降5.5% → 平局被推离（竞彩D仅-0.7%，轻微）→ **平局不打出**
- 澳门推客+客赔0%不动 → **规律F类推：澳门推客+客赔锁定=客队是真方向？**
- 近况差-3 → 客队优势
- **正确的V3.4判断应该是：客胜**

**失败原因分析**：
1. V3报告用"规律五（主升>5%）"判断平局 → 但规律五的触发条件是"主升>5%+赔率不锁定"，这里澳门A赔锁定但竞彩A降-8.2%→客队方向明确
2. 竞彩D仅降0.7% → 平局方向不明显，不能确认平局
3. **规律五在世预附加赛中失效**——淘汰赛性质比赛筹码逻辑不同

**新规律候选**：淘汰赛（世预附加赛/杯赛淘汰轮）中，规律五降级/失效。

### 周四006 捷克 vs 爱尔兰（2-2 平局）❌

**H2d模式：澳门不动** ★★★

| 维度 | 数据 | 信号 |
|------|------|------|
| 澳门方向 | H-2.6% D0 A+4.7% | 微变 |
| 竞彩方向 | H0 D0 A0 | 完全不动 |
| 澳门推荐 | 客胜 | |
| 近况差 | -1 | 几乎均衡 |

**V3报告判断**：主胜
**实际结果**：2-2 平局 ❌

**V3.4框架重新分析**：
- 注意：这里是**竞彩不动、澳门微动** → 不是"澳门不动"，是"竞彩不动"（澳门独控）
- 澳门独控：以澳门方向为准
- 澳门H降-2.6% → 降主 → 承接方向 = 主队可能打出？但幅度不大
- 澳门A升+4.7% → 推离客队 → 客队不打出
- 澳门推客+客被推离 → **澳门推的方向自己都在推离 → 信号矛盾**
- 近况差-1均衡 → 无基本面方向

- **正确的V3.4判断：信号矛盾 → 不参与**

**失败原因**：
1. V3报告标记"全锁定"但实际竞彩全不动而澳门微变 → 模式误判
2. 澳门推客+客被推离 = 典型信号矛盾 → 应标不参与
3. **澳门独控+推离自己推荐方向 = 强烈不参与信号**

**新规律候选**：澳门独控模式中，如果澳门推的方向同时被澳门推离（赔率升），则强烈不参与。
