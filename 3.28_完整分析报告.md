### 001 町田泽维 vs 川崎前锋 — 造热深度分析

**赔率格局**：主-16.4%大幅下降，客+38.4%极端推离，平+3.4%微升

**三维度检查**：
1. ✅ **幅度**：总和58.2% >> 10%
2. ✅ **出口**：主大幅降+客极端升+平微升 → **单出口主胜（筹码全涌向主胜）**
3. ✅ **同向**：澳门推主胜 + 近况差+9主优
4. **第四维度（基本面分流）**：
   - 町田泽维是日职中上游队（非超级豪门但有一定知名度）
   - 川崎前锋近期惨败0:5，状态极差
   - **筹码天然信任主胜方向** → 筹码无法有效分流

**判断**：三维度+基本面全满足 → **真造热** → 反向（主胜不出）

**但需注意**：主赔从2.01降到1.68，即时1.68是实盘中低赔付区。近况差+9也极大。

**矛盾点**：造热体系说反向，但近况差+9+主赔1.68低赔付在规律O下指向主胜。

**结论**：造热信号（反向）与规律O（近况差+9+赔率微变...不对，这里赔率大变）冲突。赔率变化58.2%说明庄家在**大幅调控**，不是"微变"场景。在大幅调控场景下，庄家主动大幅降水主赔，筹码全部涌向主胜，赔付压力极大 → **造热反向信号优先** → 主胜不打出 → **平局或客胜**。

**但近况差+9差距太大...川崎近期0:5惨败，状态极差。** 町田主场+状态碾压，即使造热，基本面差距过大也可能直接打出。

**最终判断**：001造热信号存在但基本面差距过大，**降级为观望**。
