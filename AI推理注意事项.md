Step 3: 用系统推荐作为"参考"（不是"答案"）

## 注意事项1：近况区间判断

### ⚠️ 必须先判断近况区间！

**近况区间分类**：
```
近况合计 < 3.5   → 偏低（预期小球）
近况合计 3.5-4.0 → 正常
近况合计 4.0-5.0 → 偏高（预期大球）
近况合计 > 5.0   → 极高（预期大球，可能有极端比分）
```

### 为什么重要？

**案例（巴黎FC VS 里尔）**：
```
近况合计 = 5.6（极高）
系统推荐：3球（黄金3球）❌
框架推理：5球或6球（双选）✅

错误原因：系统忽略"近况5.6极高"，仍然推荐3球
```

### 正确做法

```
Step 1: 计算近况合计 = 主近况 + 客近况
Step 2: 判断近况区间（偏低/正常/偏高/极高）
Step 3: 根据近况区间，选择不同的推理策略
  - 偏低 → 重点考虑0-2球
  - 正常 → 重点考虑2-3球
  - 偏高 → 重点考虑3-4球
  - 极高 → 重点考虑4-7球 + 触发5.8节（极端情况预警）
```

## 注意事项2：大球区vs小球区

### ⚠️ 大球区和小球区的"赔率不变"含义完全不同！

（这是用户重大洞察！2026-04-27）

```
大球区（4-7球）赔率不变 = 正常！✅
  → 原因：赔率高 + 投注拉力少 + 庄家不需要调整
  → 不能因为"变化0次"就排除大球

小球区（0-3球）赔率不变 = 异常！⚠️
  → 原因：赔率低 + 投注拉力多 + 庄家应该调整
  → 可能是"庄家故意不设防"（诱盘）
```

### 案例（帕德博恩 VS 沙尔克04）

```
近况：主3.6 + 客2.6 = 6.2（极高）

大球区：
  4球：变化0次 → 正常！✅ 不能排除4球
  5球：变化0次 → 正常！✅ 不能排除5球

小球区：
  2球：变化0次 → 异常！⚠️ 可能是诱盘
  3球：变化0次 → 异常！⚠️ 可能是诱盘
```

### 正确做法

```
如果 变化0次：
  先判断是"大球区"还是"小球区"
  
  如果是大球区（4-7球）→ 正常！不要排除 ✅
  如果是小球区（0-3球）→ 异常！⚠️ 可能是诱盘
```

## 注意事项3：赔率跳跃诱导

### ⚠️ 当大球区出现"赔率跳跃"时，低赔率侧是"高诱导区"！

（这是用户重大洞察！2026-04-27）

```
正常情况（无诱导）：
  4球4.65 → 5球6.5 → 6球10.0 → 7球16.0
  → 赔率"平缓上升" → 无诱导 ✅

异常情况（有诱导）：
  4球4.65 → 5球8.25（跳跃！）⚠️
  → 赔率"突然跳跃" = 庄家诱导信号 ⚠️
```

### 为什么是"高诱导区"？

**玩家心理**：
```
1. 看到近况极高（>5.0）→ "肯定是大球！"
2. 看到4球4.65（相对低）→ "4球最有可能！"
3. 看到5球8.25（突然高）→ "5球不可能！"

→ 集中买4球
→ 实际打出5球（或6球）
→ 庄家通吃！✅
```

### 识别方法

```
如果 近况合计 > 4.0（偏高或极高）
  + 4球赔率 < 5.0
  + 5球赔率 > 8.0（突然跳跃！）
  
  → 4球是"高诱导区" ⚠️
  → 不要买4球！
  → 重点考虑5球（或6球）✅
```

**量化标准**：
```
跳跃幅度 = (5球赔率 - 4球赔率) / 4球赔率
如果 跳跃幅度 > 50%（如4.65→8.25，跳跃77%）
  → 强诱导信号！⚠️
```

## 注意事项4：大球区过度防范陷阱

### ⚠️ 当大球区"过度防范"时，可能是庄家陷阱！

（这是用户重大洞察！2026-04-27）

```
正常情况（真实防范）：
  近况极高（>4.0）→ 庄家防范大球 → 4-7球小幅下降（5-15%）
  
异常情况（过度防范=陷阱）：
  近况极高（>4.0）→ 庄家"过度防范"大球 → 6-7球大幅下降（>20%）⚠️
  → 庄家故意大幅降低大球赔率，诱导玩家买大球
  → 实际可能打出小球（0-2球）✅
```

### 案例（巴黎FC VS 里尔）

```
近况：主3.2 + 客2.4 = 5.6（极高！）

大球区：
  4球：变化3次 ↓7.8%（正常）
  5球：变化3次 ↓11.1%（正常）
  6球：变化3次 ↓23.5%（降幅极大！）⚠️
  7球：变化4次 ↓32%（降幅极大！）⚠️
  
→ 大球区"过度防范"陷阱！⚠️
→ 实际打出1球（小球）✅
→ 庄家完美诱导！✅
```

### 识别方法（✅ 已验证，2026-04-27更新）

````
如果 近况合计 > 4.0（偏高或极高）
  + 6球或7球降幅 > 20%（如6球↓23.5%，7球↓32%）
  → 5.13节触发！

  【⚠️ 关键：必须用变化次数判断是否成立！】

  如果 大球区变化次数 ≥ 4次
    → "招摇陷阱"成立！→ 推荐小球（0-3球）
    → 案例：切尔西 VS 利兹联（5次变化，1球✅）

  如果 大球区变化次数 ≤ 2次 且 降幅 < 15%
    → 5.13节不成立！→ 5.8节优先 → 推荐大球（5-7球）
    → 案例：莫尔德 VS 瓦勒伦加（1次变化，6球✅）

  如果 大球区变化次数 ≤ 2次 且 降幅 > 20%
    → "可能陷阱" → 推荐小球
    → 案例：巴黎FC VS 里尔（1次变化，1球✅）
````

### 🆕 变化次数 = "招摇程度"（2026-04-27发现，3/3验证）

| 类型 | 变化次数 | 降幅 | 含义 | 推荐 |
|------|---------|------|------|------|
| **真实防范** | **1-2次** | **<15%** | 庄家"悄悄"设防，不想引人注意 | **大球** |
| **可能陷阱** | **1-2次** | **>20%** | 变化少但力度大，可能诱导 | **小球** |
| **招摇陷阱** | **4-5次** | **>20%** | 庄家"高调"防范，引人注目 | **小球** |

**核心逻辑**：
- 庄家越"低调"的防范越真实 → 推荐大球
- 庄家越"高调"的防范越可能是陷阱 → 推荐小球
- 变化次数是"招摇程度"的最佳指标

### 正确做法

````
如果 近况合计 > 4.0
  + 大球区（4-7球）降幅极大（>20%）

  第一步：检查变化次数
  第二步：
    变化次数 ≥ 4次 → 推荐小球（招摇陷阱）⚠️
    变化次数 ≤ 2次 且 降幅 < 15% → 推荐大球（真实防范）✅
    变化次数 ≤ 2次 且 降幅 > 20% → 推荐小球（可能陷阱）⚠️
    变化次数 = 3次 → 不确定，需要其他信号辅助
````

### ⚠️ 当5.8节和5.13节冲突时

```
5.8节（极端情况预警）推荐大球
5.13节（过度防范陷阱）推荐小球

→ 用变化次数判断：
  变化次数 ≤ 2次 → 5.8节优先（真实防范大球）
  变化次数 ≥ 4次 → 5.13节优先（招摇陷阱，推荐小球）
```

## 注意事项6：二选一推理方法优先级

### ⚠️ 当4个推理方法不一致时，需要优先级规则！

```
方法1（赔率高低）：推荐最低赔率的选项
方法2（隐含概率）：推荐隐含概率最高的选项
方法3（变化方向+幅度）：推荐下降的选项
方法4（让球盘辅助）：推荐让球盘支持的总进球数
```

### 问题：4个方法可能不一致！

**案例（SBV精英 VS 乌德勒支）**：
```
方法1：推荐5球（5球赔率8.0最低）
方法2：推荐5球（5球隐含概率12.5%最高）
方法3：推荐6-7球（不是5球）⚠️ 不一致！
方法4：支持5球（让胜1.67超低赔）

→ 方法1/2推荐5球，方法3推荐6-7球 → 不一致 ⚠️
```

### 优先级规则（建议）

```
第1优先级：5.8节（极端情况预警）✅
  → 如果触发，直接忽略方法1/2的推荐

第2优先级：方法3（变化方向+幅度）✅
  → 庄家真实态度，最可靠

第3优先级：方法4（让球盘辅助）✅
  → 让球盘提供重要参考

第4优先级：方法1/2（赔率高低/隐含概率）❌
  → 静态赔率，可能误导
```

## 注意事项7：充分利用让球盘

### ⚠️ 让球盘可以提供重要参考！

**让球盘的含义**：
```
让球(-1)：主队让1球
  → 让胜：主队净胜2球以上
  → 让平：主队净胜1球
  → 让负：主队不胜（平或负）

让球(+1)：主队受让1球
  → 让胜：主队不败（胜或平）
  → 让平：主队1球小负
  → 让负：主队2球以上大败
```

### 案例（莫尔德 VS 瓦勒伦加）

```
让球(-1)：让胜3.34 / 让平3.92 / 让负1.74

让负1.74超低赔 → 庄家看好"主队让1球后客队不败"
= 主队可能大胜（3-0, 3-1, 4-0, 4-1, 4-2, 5-1...）
= 总进球可能很多（4+球）✅
```

### 正确做法

```
Step 1: 读取让球盘赔率
Step 2: 找出超低赔率（<2.0）的选项
Step 3: 解读庄家态度（看好哪一方？）
Step 4: 推导可能的总进球数范围
Step 5: 与"二选一推理方法"的结果对比，取交集
```

```
Step 1: 自己计算近况合计 = 主近况 + 客近况

Step 3: 判断近况区间（偏低/正常/偏高/极高）
Step 4: 不要相信系统的"近况均值"显示，要自己复核 ⚠️

## 注意事项10：变化方向+幅度分析

### ⚠️ 只看静态赔率是不够的！必须分析"变化方向+幅度"！

**静态赔率 vs 变化方向+幅度**：
```
静态赔率：3球3.4（最低）→ 玩家最想投注3球
变化方向：3球变化0次 → 庄家不设防 ⚠️
变化幅度：无（变化0次）

→ 静态赔率低 ≠ 庄家看好
→ 变化方向+幅度才反映庄家真实态度 ✅
```

### 案例（帕德博恩 VS 沙尔克04）

```
静态赔率：
  2球3.55（最低）→ 玩家最想投注2球
  3球3.5（次低）→ 玩家次想投注3球
  
变化方向+幅度：
  2球变化0次 → 庄家不设防 ⚠️ 可能是诱盘
  3球变化0次 → 庄家不设防 ⚠️ 可能是诱盘
  4球变化0次 → 庄家不设防 ✅ 正常（大球区）
  5球变化0次 → 庄家不设防 ✅ 正常（大球区）
  
→ 实际打出5球（不是2-3球）✅
→ 静态赔率误导！❌
```

### 正确做法

```
Step 1: 读取各进球数的"变化次数"和"变化方向+幅度"
Step 2: 判断庄家态度（看好/不看好/不设防）
Step 3: 排除"庄家不看好"的选项（上升=不看好）
Step 4: 谨慎对待"庄家不设防"的选项（可能是诱盘）

### 错误案例2：忽略"大球区vs小球区"区别

```

```
