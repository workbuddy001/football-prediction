## 三、教训2：变化命中率阈值守护（核心！核心！核心！）

### 我的错误排除

```
错误分析:
  1球: 赔率10.5>8.0 + 赔率命中率0%(1场) → 🚫 排除1球
  结论: 排除1球后，2球和3球保留 → 推荐3球 ❌

### 1球凭什么不能被排除

| 维度 | 数据 | 阈值 | 判定 |
|------|------|------|------|
| 变化命中率 | **15.8%** (19场) | ≥**15%** | ⚠️⚠️⚠️ 强保留！ |

| 赔率 | 10.50 | >8.0 | 偏高，弱信号 |

**变化命中率 15.8% ≥ 15% + 样本 19 ≥ 10 → 🚫 绝对不能排除！**

我的错误：被"赔率=10.5>8.0 + 系统排除1球"误导，忽视了**变化命中率是终极裁判**。

### 正确分析路径

```

## 四、变化命中率阈值规则体系

```
≥30% + 样本≥10 → 超级保留（推翻任何排除）
≥25% + 样本≥10 → 强保留
≥15% + 样本≥10 → 保留（不可推翻！）← 本场: 1球15.8%
≥10% + 样本≥10 → 弱保留（需要至少一个其它支持信号）
<10% 或 样本<10 → 可排除（需结合其它信号）
```

### 排除失败的常见模式

| 错误排除 | 正确做法 |
|---------|---------|
| 赔率高→排除 | 先看变化命中率：≥15%→不能排除 |
| 赔率命中率0%→排除 | 先看样本量：<5→不参与排除 |
| 系统排除规则→排除 | 先看变化命中率：≥15%→推翻系统排除 |
| 被推离(↑)→排除 | 先看变化命中率：≥30%→假推离→保留 |

## 七、排除法三维决策框架（本场实战总结）

> 每个候选进球数从三个维度综合判断：**热度 / 命中率 / 排除信号**。三维交叉决定排除还是保留。

### 7.1 三维决策表

| 进球 | 双命中率 | 变化方向 | 热度 | 系统排除 | 判断 |
|------|---------|---------|------|---------|------|
| **3球** | 28.1%+26.1%=**双高≥20%** | ↓防 | 🔥**大热** | 有(三条件) | 🚫 排除 |
| **1球** | 15.8%+0%≠双高 | ↑推 | 🌊**无热** | 有(>8.0) | ✅ **保留** |
| **4球** | 8.6%+0%=**双低** | ↓防 | 🌊无热 | 有(>30) | 🚫 排除 |

### 7.2 决策规则

```
排除成立（同时满足）:
  ① 双命中率 ≥ 双高(都≥20%) 或 双低(都<10%)
  ② 热度方向明确（双高=大热/双低=无热）
  ③ 系统有排除信号 或 变化方向配合

保留成立（满足任一）:
  ① 变化命中率 ≥ 15% → 不管赔率多高、系统怎么排除，保留
  ② 双命中率双高但无系统信号 → 警惕造热，但不自动排除
  ③ 任一命中率 ≥ 10% 且 另一维度样本<5 → 不排除
```

### 7.3 决策矩阵

```
                   变化命中率
                   高(≥25%)    中(15-25%)   低(<10%)
赔率命中率  ┌─────────────────────────────────────┐
高(≥20%)    │ 大热+双高   大热+变中    矛盾+排除    │
            │ → 有排除=排除 → 警惕造热   → 排除     │
中(10-20%)  │ 变高共振    保留         弱排除      │
            │ → 强保留    → 保留       → 弱排除    │
低(<10%)    │ 矛盾保留    保留         🚫 双低      │
            │ → 保留      → 保留       → 排除      │
样本不足(<5)│ 变高保留    保留         不参与      │
            │ → 保留      → 保留       → 保留      │
└─────────────────────────────────────────────────┘
```

### 7.4 本场三例对照

**3球 — 大热必死型**:
```
双高(28.1%+26.1%) + 赔率↓(防范=制造"稳"的错觉)
+ 系统三条件排除 + 初盘3.80>3.50
→ 四个维度全指向3球 = 太完美 = 造热
→ 🚫 排除正确
```

**1球 — 温吞保留型**:
```
变中(15.8%) + 赔率0%(样本1不参与) + 赔率↑(推离)
+ 赔率10.5(高赔=庄家不看好) + 无热度
→ 变化命中率≥15%是唯一强信号，但足够打破排除
→ ✅ 保留正确（实际0-1打出）
```

**4球 — 双低排除型**:
```
双低(8.6%+0%) + 赔率↓(表面防范但命中率不支撑)
+ 系统排除(0球>30) + 赔率=3.80(不高但0%(3场))
→ 变化8.6%<10% + 赔率0%(样本3<5不参与) = 双低
→ 🚫 排除正确
```
