### 规律审核详解

**周日006 科莫 vs 比萨** ⭐⭐⭐首选稳胆
- 置信度75.7%（全场最高）
- 澳门推荐主胜，同向
- 近况差+10（主队碾压）
- 赔率实盘无变化
- **规律G判定**：赔率不变+近况差大→大胜格局（如3月21日本菲卡3-0）
- **结论**：主胜稳胆，可能大胜

**周日022 罗马 vs 莱切** ⭐⭐⭐稳胆
- 置信度68.1%
- 澳门推荐主胜，同向
- 赔率实盘无变化
- **规律一判定**：≥66%+澳门同向+实盘→主胜可打出
- **结论**：主胜稳胆

**周日001 首尔FC vs 光州FC** ⭐⭐较稳
- 置信度66.1%（刚达门槛）
- 澳门推荐主胜，同向
- 主降6.9%（庄家真实看好）
- 近况差+4
- **规律一判定**：≥66%+澳门同向+主降→主胜
- **结论**：主胜较稳

**周日012 亚特兰大 vs 维罗纳** ⭐⭐热度分散
- 置信度69.2%
- 澳门推荐**和局**（非主胜）
- 近况差=0（双方均衡）
- 赔率变化：主+0.8%/平-1.1%/客-2.5%（幅度均<5%）
- **规律H判定**：
  - ①澳门推平局→主队热度被分散
  - ②近况差=0→战绩角度平局合理
  - ③赔率变化小→市场稳定无陷阱
  - → **没有热度的高置信度比赛，按置信度判断，主胜打出**
- ⚠️ 脚本原修正为"客胜"是误判（因触发规律D，但规律D适用于64-65%且造热）
- **结论**：按规律H修正为**主胜**

### 规律审核详解

**周日021 圣保利 vs 弗赖堡** ⭐⭐⭐爆冷首选
- 置信度34.7%（极低）
- 澳门推荐和局
- 平局初始赔率2.89**<3.0**
- **规律二判定**：平赔<3.0→平局难出，排除和局
- 近况差=0，主队无优势
- **规律三判定**：低置信度，排除和局后按赔率方向
- **结论**：排除平局 → **客胜**

**周日003 浦和红钻 vs 町田泽维** ⭐⭐⭐爆冷首选
- 置信度35.1%（极低）
- 澳门推荐和局
- 平局初始赔率2.95**<3.0**
- **规律二判定**：平赔<3.0→平局难出，排除和局
- 近况差-2，客队稍好
- **结论**：排除平局 → **客胜**

**周日002 大阪樱花 vs 神户胜利** ⭐⭐⭐极端造热
- 置信度38.8%（异常低）
- 澳门推荐客胜
- 近况差-10（客队碾压）
- 赔率变化：主-14.1%（极端） 平不变(3.35) 客+12.3%（极端）
- **规律I判定**：
  - ①澳门推客 + 近况差极大(-10)
  - ②赔率极端：主降>10% + 客升>10%
  - ③置信度异常低（与近况差不匹配）
  - ④平局赔率不变(3.35)，中庸不吸筹码
  - → **极端造热双向对冲 → 平局**
- **结论**：主队打不出+客队过热打不出 → **和局**

**周日011 博洛尼亚 vs 拉齐奥** ⭐⭐实盘客胜
- 置信度40.3%
- 澳门推荐和局
- 平局初始赔率2.95**<3.0**
- 近况差-2，客队稍好
- 赔率变化：主+4.8% 平-3.4% 客-2.5%
- **规律J判定**：
  - ①澳门推平局（平局最热）
  - ②平赔<3.0→规律二排除平局
  - ③主队抬水4.8%→主胜难
  - ④客队降水2.5%进入2.X区间→实盘看好
  - → **澳门推平+平赔低+客悄悄降水 → 客胜**
- **结论**：排除平局+主升+客降 → **客胜**

**周日017 格罗宁根 vs 阿尔克马** ⭐⭐⭐极端造热主胜冷门
- 置信度43.2%（异常低）
- 澳门推荐客胜
- 近况差-10（客队碾压）
- 赔率变化：主+11.2% 平+2.9%(3.45→3.55) 客-10.1%
- **规律L判定**：
  - ①澳门推客 + 近况差极大(-10)
  - ②赔率极端：主升>10% + 客降>10%
  - ③置信度异常低（与近况差不匹配）
  - ④平赔**不降反升**（3.45→3.55）
  - → **极端造热+平赔反升 → 排除客胜+排除平局 → 主胜**
- **结论**：客队过热+平赔反升 → **主胜（冷门）**

**周日015 美因茨 vs 法兰克福** ⭐⭐强造热主不败
- 置信度42.2%
- 澳门推荐客胜
- 近况差-1（双方持平）
- 赔率变化：主+9.9% 平-5.8% 客-8.2%
- **规律K判定**：
  - ①澳门推客 + 近况差接近(-1)
  - ②客队降水>8%（强造热）
  - ③主队升水>8%
  - → **强造热客队+近况持平 → 主队不败**
- **结论**：客队过热+主升过大 → **主胜或平局（偏主胜）**

**周日026 佛罗伦萨 vs 国际米兰** ⭐⭐主升过大
- 置信度60.7%
- 澳门推荐客胜
- 近况差+7（主队好）
- 赔率变化：主+12.9% 平+4.8% 客-5.2%
- **规律五判定**：主胜升幅>5%→平局概率大
- **问题**：主队近况好但赔率大幅抬升 → 异常
- **结论**：主升>12%（规律五）→ **和局**

**周日027 皇马 vs 马竞** ⭐德比实盘
- 置信度53.1%
- 澳门推荐和局
- 近况差+6（皇马好）
- 赔率变化：主-4.6% 平+1.3% 客+8.7%
- **规律M判定**：
  - ①马德里德比，投注量巨大
  - ②赔率变化幅度均<10%（非极端）
  - ③主降4.6%+客升8.7%→实盘看好主队
  - → **德比战庄家无法造热→按实盘方向**
- **结论**：主降客升，实盘方向 → **主胜**
