# 足球比分推理框架 V3.6 — 完整流程说明

> 基于排除法思想，通过十步推理从大小球方向到最终比分推荐。
> 核心原则：**变化命中率为尊，Step0为臣。盘口是表面，赔率变化结构是真相。**

---

## ⚠️ 重要前置认知：方向不是一次性判断

**Step0 的大小球方向只是"初步判断"，不是最终结论。**

随着后续步骤的深入，方向可以被纠正甚至反转：

```
Step0 初步方向 → Step2.5 造热检查 → Step4 排除法命中率校验 → 最终方向

三个纠正节点:
  ① 5.24节 变化命中率否决权: 方向内数据全不达标 → 反转方向
  ② 5.23节 H/I模式: 盘口表面≠赔率变化真相 → 反转方向
  ③ Step4 排除后单侧空空: 该方向无进球可保留 → 重新审视方向
```

**今日五场实战中的方向纠正率：**

| 比赛 | Step0方向 | 纠正节点 | 最终方向 | 结果 |
|------|----------|---------|---------|------|
| 赫尔辛基vs拉赫蒂 | 大球 | → ②H模式 | **小球** | 1-0 ✅ |
| 哈尔姆斯vs布鲁马 | 小球 | → ②I模式 | **大球** | 1-3 ✅ |
| 博德闪耀vs莫尔德 | 大球 | → ①否决权 | **小球** | 0-1 ✅ |
| 罗马vs佛罗伦萨 | 不确定 | → ③单侧空 | **大球** | 4-0 ✅ |
| 克雷莫纳vs拉齐奥 | 小球 | 不变 | 小球 | 1-2 ❌→3球✅ |

**教训**：Step0 只是一个起点。如果 Step0 错了而后续分析不留纠正空间，整个推理链条必然错误。**V3.6 的核心设计就是层层校验、允许反转。**

## 流程总览

```
Step 0  大小球方向判断（四要素 → 实防/造热/模糊）
   ↓               ← 5.24节校验：变化命中率否决权
Step 2.5 造热检查（五信号 → 排除被造热方向）
   ↓
Step 4  排除法 + 双命中率交叉验证（三维决策矩阵）
   ↓               ← 4.7节三维排除：热度/命中率/排除信号
Step 5  特殊规则检查（5.8-5.24节）
   ↓
Step 6  胜平负/让球盘辅助
   ↓
Step 7.8 比分反推验证（V3.6能力评估法）
   ↓
Step 7.9 终审：意料之外、情理之中 ← ⚠️ 反直觉! 可能颠覆结论
   ↓
Step 7.10 反审：赔付压力自检 ← ⚠️ 迎合+压力=陷阱! 可能否定结论
   ↓
最终推荐：总进球数 + 比分 + 胜平负 + 大小球
```

## Step 0：大小球方向判断（V3.6核心·优先级最高）

### 输入四要素

| 要素 | 来源 | 示例 |
|------|------|------|
| 近况合计 | 主+客近5场场均 | 主2.2+客3.6=5.8 |
| 系统标准线 | 查表(近况→标准线) | 5.8→3.125 |
| 亚盘大小球+水位 | 用户手动录入 | 大0.77/2.75/小0.93 |
| 0球赔率 | 总进球数据 | 16.0 |

### 三种信号源

```
0球方向: 实际0球 vs 理论区间[主+10, 客+10]
  偏差>+2→大球 | 偏差<-2→小球 | 居中→中性

线位方向: 实际线位 vs 标准线
  差>+0.5→大球 | 差<-0.5→小球 | 居中→中性

水位方向: 低水<0.85 → 低水侧为方向
```

### 六种判断结果（V3.6）

| 模式 | 条件 | 结论 | 案例 |
|------|------|------|------|
| **A** 三信号共振 | 0球+线位+水位 全同向 | 🛡️ 该方向强实防 | 科莫0-0 |
| **B** 双信号 | 0球+线位 或 线位+水位 同向 | 🛡️ 该方向实防(83%) | 乌迪内斯2-0 |
| **D** 深线确认 | 线位>标准+0.5 + 大球低水 | 🛡️ 大球 | 勒沃库森4-1 |
| **H** 线位偏浅造热大球 | 线位<标准-0.3+大球低水+0球↓+高球↑推离 | 🔥 小球! | 赫尔辛基1-0 |
| **I** 表面小球造热 | 线位偏浅+小球低水+0球→0%+小球变化全差 | 🔥 大球! | 哈尔姆斯1-3 |
| **模糊** | 信号矛盾/不足 | 🌫️ 全范围分析 | — |

### H/I模式区分关键

```
表面大球→小球(H): 0球↓防范  ← 庄家怕小球
表面小球→大球(I): 0球→0%不防 ← 庄家不怕小球
```

## V3.6 核心：变化命中率否决权（5.24节）

**Step0输出方向 ≠ 最终方向。必须校验变化命中率。**

```
校验1: 0球防御检测
  Step0=小球 + 0球→0% → 庄家不防小球 → -30%

校验2: 方向内变化命中率检测
  方向内进球变化命中率全<15%  → -40%
  方向内≥2球变化=0%           → 否决！方向不成立

降权≥50% → 方向反转
降权30-50% → 全范围分析，不以Step0过滤
降权<30% → Step0方向可信
```

## Step 4：三维排除决策（4.7节）

每个候选进球数从三个维度判断：**热度 / 命中率 / 排除信号**。

### 决策矩阵

```
                   变化命中率
                   高(≥25%)    中(10-25%)   低(<10%)
赔率命中率  ┌─────────────────────────────────────┐
高(≥20%)    │ 🔥大热+双高   🔥热+变中    ⚡矛盾     │
            │ → 有排除=排除  → 警惕造热   → 排除    │
中(10-20%)  │ ⭐变高共振    ✅保留       → 弱排除   │
            │ → 强保留                            │
低(<10%)    │ 🔄矛盾保留    ✅保留       🚫 双低    │
            │ → 保留                   → 排除     │
└─────────────────────────────────────────────────┘
```

### 排除黄金法则

```
1. 变化命中率≥15%+样本≥10 → 🚫 绝对不可排除！
   推翻任何赔率高低、系统排除、推离信号

2. 双高(都≥20%)+赔率↓+系统排除 → 🔥大热必死 → 排除

3. 双低(都<10%)+系统排除 → 🚫 排除

4. 赔率命中率=0%+样本<5 → ⛔ 不参与排除判断
```

### 实例：博德闪耀0-1莫尔德

```
3球: 28.1%+26.1%=双高 + 赔率↓ + 三条件排除 → 🔥大热必死 → 🚫排除
1球: 15.8%≥15% + 赔率↑推离 + 无热度 → ✅铁保留 → 实际0-1 ✅
4球: 8.6%+0%=双低 + 系统排除 → 🚫排除
```

## Step 7.8：比分反推验证（V3.6能力评估法）

### 旧规则 vs 新规则

```
❌ 旧(V3.5): 近5场是否打出过N球？→ 没有 → 排除
✅ 新(V3.6): 攻击力是否足够？防守是否支持？
```

### 能力评估三指标

| 指标 | 计算方法 | 阈值 |
|------|---------|------|
| **攻击力** | 近5场场均进球 + 峰值经验 | 场均≥所需×0.5 且 有相近峰值(差≤1) |
| **防守力** | 近5场场均失球 | 失球≤1=铁壁 | 1-2=中等 | >2=脆弱 |
| **零封概率** | 客队攻击力反向验证 | 客场均<1.0 → 零封概率高 |

### SOP

```
for 总进球 in [首选, 备选, ...]:
    比分列表 = 所有可能比分 → 让球盘过滤
    
    for 比分 in 比分列表:
        C1: 赔率热度>15%? → 过热排除
        C2: 主队攻击力足够?
        C3: 客队防守是否支持?
        C4: 客队攻击力→零封概率?
        → 通过 → 可行
    
    有可行比分? → 推荐 → break
    无可行比分? → 退到下一总进球
```

### 实例：罗马4-0佛罗伦萨

```
4球 → 4-0, 3-1, 2-2, 1-3, 0-4
主胜1.42 → 4-0, 3-1

4-0验证:
  C1: 热度不高 ✅
  C2: 场均1.8球 + 3-0经验 → 攻击力足够 ✅
  C3: 场均失0.8球(中) → 可被进多球 ✅
  C4: 客队场均0.8球(极弱) → 零封概率高 ✅
  → 4-0可行 ✅ 实际4-0 ✅
```

## 完整实例1：赫尔辛基1-0拉赫蒂（H模式·小球方向）

### 数据

```
近况: 2.2+3.6=5.8 | 标准线: 3.125
盘口: 大0.77/2.75/小0.93 | 0球: 16.0
```

### Step0：表面判大球，变化反转

```
信号: 0球(大球) + 线位(中性) + 水位(大球) → 2/3大球
H模式校验: 线位2.75<3.125(偏浅) + 0球↓5.9% + 6球↑14.3%/7球↑10%
→ 盘口做给你看大球，赔率变化指小球 → 🔥方向反转 → 小球
```

### 排除法

```
1球: 系统排除(0/23) → 🚫
3球: 五信号全指向 → 🔥造热排除
2球: ↓1.3% + 21.1%(19)+10%(10) → 首选
```

### 比分反推

```
2球: 2-0(主队无2球能力❌), 1-1(不支持❌), 0-2(方向❌) → 全军覆没
→ 退1球: 1-0 ✅

最终: 1球/1-0/主胜 → 实际1-0 ✅
```

## Step 7.9：终审——意料之外、情理之中（V3.6新增）

> **框架推理的最后一步。跳出框架思维，问：如果全错了，最可能的意外是什么？**

### 终审三问

```
① 破格攻击力: 主队（或客队）是否有能力打出远超近况的爆发性比分？
   → 让球盘深度(≥2球) + 主胜极低(<1.30) + 历史峰值

② 对手脆弱性: 对手防守是否存在崩盘风险？
   → 近5场被同级/高级3+球击败的记录

③ 最强变化信号: 全场最强赔率变化指向什么方向？
   → 变化幅度+次数+命中率综合
```

### 实例4：里斯本竞技5-1吉马良斯

```
我的预测: 2球/2-0 ❌ | 实际: 5-1 (6球!)

终审三问:
① 让球-2 + 主胜1.25 → 破格攻击力 YES
② 吉马良斯 0-3 本菲卡(同级) → 脆弱性 YES
③ 4球↓11.4%(3次)全场最强 + 3球↓5.7% → 大球信号 YES

三问全YES → 🚨框架保守! 添加意外备选
→ 常规:2-0 | ⚠️意外大胜:4-1/5-1 → 实际5-1 ✅
```

## Step 7.10：反审——赔付压力自检

> **最后一步：如果结论太"完美"——基本面、盘口、水位全迎合——庄家在等你上钩。**

### 核心逻辑

```
"明牌"结论:
  近况好 + 看好大球 + 盘口大球低水 + 赔率获推荐
  → 玩家大量投注 → 庄家赔付压力巨大
  → 庄家不会让"明牌"轻松打出 → 实际反向
```

### 反审三问

```
① 迎合检查: ≥2项即成立
  近况方向 = 结论方向？
  大小球水位 = 结论方向？
  让球盘方向 = 结论方向？

② 赔付压力检查: 任一即成立
  结论进球赔率 < 4.0（低赔"明牌"）？
  双命中率 ≥ 20%（投注集中）？
  赔率↓（表面防范=造热投注）？
  被系统推荐（关注度高）？

③ 矛盾判断:
  迎合 + 压力 → 🚨 结论是陷阱! 降置信两档
  迎合 only → ⚠️ 降一档
  两者皆无 → ✅ 安全
```

### 四象限决策表

| 迎合 | 赔付压力 | 结论 | 操作 |
|------|---------|------|------|
| ✅ | ✅ | 🚨陷阱 | 降置信两档，寻找"不得不赔"理由 |
| ✅ | ❌ | ⚠️可疑 | 降一档，检查隐藏信号 |
| ❌ | ✅ | ⚠️边缘 | 正确但被市场低估 |
| ❌ | ❌ | ✅安全 | 结论可信 |

### 特殊排除规则：双高+推离+高赔 >3.90

```
双命中率 ≥ 20% + 赔率↑推离 + 赔率 > 3.90
→ 🚨 反审特殊排除!

逻辑:
  统计说能打出(双高) + 庄家推离(↑) + 高赔付(>3.90)
  → 庄家不仅不怕你买，还给你高赔率
  → 这不是机会，是诱饵
  → 该进球数反审排除
```

### 实例：里斯本竞技5-1吉马良斯

```
3球检查:
  ① 迎合: 近况大球 + 盘口大球低水 = 2项 → 迎合 ✅
  ② 压力: 3.30∈[3.2,3.5]舒适 + 双高30.3%+20% + ↓5.7% + 系统推荐
     → 四项全满足 → 压力 🚨
  ③ 迎合+压力 → 陷阱! → 造热排除 ✅

2球检查:
  ① 迎合: 近况方向? 盘口? → 仅1项 → 不触发 ✅
  ③ 结论安全 ✅

实际: 3球被造热排除 → 正确! 2球推荐安全但实际大胜5-1
→ 还需7.9终审补充
```
