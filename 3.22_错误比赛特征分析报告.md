## 三、错误比赛主要特征

### 特征1：规律五失效严重（最大问题）

**规律五**：主胜升幅>5% → 预测和局

| 场次 | 主胜变化 | 预测 | 实际 | 结果 |
|------|----------|:----:|:----:|:----:|
| 周日004 | +15.2% | 平局 | 1:1 | ✅ |
| 周日011 | +8.2% | 平局 | 0:2客 | ❌ |
| 周日016 | +12.2% | 平局 | 3:4客 | ❌ |
| 周日017 | +18.4% | 平局 | 3:0主 | ❌ |
| 周日022 | +7.9% | 平局 | 1:0主 | ❌ |
| 周日026 | +8.0% | 平局 | 1:1 | ✅ |
| 周日027 | +9.3% | 平局 | 3:2主 | ❌ |

**规律五命中率：2/7 = 28.6%**

**失效原因分析**：
```
规律五的问题：只看主胜升幅>5%，没看其他方向的变化

周日011：主+8.2%，但客-6.5%（客也被造热）→ 客胜打出
周日016：主+12.2%，但客-15.4%（客更热）→ 客胜打出  
周日017：主+18.4%，但客-12.7%（客也热）→ 主胜打出（意外）
周日022：主+7.9%，但客-15.8%（客更热）→ 主胜打出（意外）
周日027：主+9.3%，但客-10.0%（客更热）→ 主胜打出（意外）
```

**核心问题**：当主胜升幅>5%时，如果**客胜也在降**（被造热），规律五就失效了！

## 五、改进建议

### 1. 紧急修复规律五

**当前**：主胜升幅>5% → 和局

**建议改为**：
```
主胜升幅>5% 且 客胜变化>-5% → 和局
主胜升幅>5% 且 客胜变化<-5% → 客胜（客队也被造热，规律五失效）
```

### 2. 低置信度统一观望

**当前**：低置信度也给预测方向

**建议**：
```
置信度<55% → 不推荐方向，标注"观望"
置信度55-65% → 可预测，但标注"高风险"
置信度≥66% → 正常预测
```

### 3. 增加德比战标签

增加德比战/关键战识别，应用不同规律：
```
if 德比战:
    降低规律五权重
    增加临场赔率权重
```

### 4. 优化澳门解析

增加更多球队别名映射：
- 特羅素 → 特罗姆瑟
- 國際邁亞密 → 迈阿密国际
- 巴塞羅那 → 巴塞罗那

### 5. 多重规律权重机制

周日017同时触发4条规律，但规律五优先级最高导致错误：
```
规律五：平局（90%信心）
规律N：主胜（85%信心）
规律R：主胜（85%信心）
规律L：主胜（75%信心）
```

**建议**：当多条规律冲突时，采用投票机制而非优先级。

## 六、关键结论

1. **高置信度(≥66%)依然可靠**：3场全中，100%命中率
2. **规律五需要紧急修复**：当前28.6%命中率不可接受
3. **低置信度场次应放弃预测**：18.8%命中率不如抛硬币
4. **德比战需要特殊处理**：常规规律在德比战中失效
5. **规律R已验证有效**：周日017虽然最终预测错误，但规律R判断正确（真造热→反向主胜）
