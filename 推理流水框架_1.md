## ⭐ Step 0：大小球方向判断（V3.1核心模块·推理第一步）

> **这是整个推理框架的地基。方向判断错误，后续排除法再精细也是徒劳。**
> 用四要素（近况、系统大小球、亚盘大小球盘口水位、0球赔率）判定三种状态：实盘防御 / 假盘造热 / 模糊地带。

### 0.1 输入四要素

| # | 要素 | 来源 | 示例 |
|---|------|------|------|
| 1 | 球队近况 | 近5场场均进球合计 | 主3.6+客3.4=7.0 |
| 2 | 系统大小球盘口 | 系统计算→标准线参照 | 近况3.0→2.5线 / 5.0→3.0线 / 7.0→3.5线 |
| 3 | 亚盘大小球盘口+水位 | 用户手动输入 | 大0.91/3.5/小0.79 |
| 4 | 0球赔率 | 总进球数据 | 32.00 或 9.75 |

**标准线参照**：

| 近况 | 参考标准线 |
|------|----------|
| <3.0 | 2.0-2.25 |
| 3.0-4.0 | 2.5-2.75 |
| 4.0-5.0 | 2.75-3.0 |
| 5.0-7.0 | 3.0-3.25 |
| >7.0 | 3.5+ |

### 0.2 三大判断结果

#### 🛡️ 一、实盘防御（Real Defense）

**定义**：两个及以上信号同向，且不存在矛盾信号。庄家真实防范该方向。

```
特征：
  0球方向 ← 大小球线位方向 ← 水位方向
  （至少两个一致，无矛盾）
```

**子类型（按信号多寡排序）**：

| # | 模式 | 条件 | 信号强度 | 案例 |
|---|------|------|---------|------|
| A | 三信号共振 | 0球方向 = 线位方向 = 低水方向 | ⭐⭐⭐⭐⭐ | 科莫0-0 |
| B | 0球+线位共振 | 0球明确偏离 + 线位同向 | ⭐⭐⭐⭐ | 科林蒂安2-0 |
| C | 浅线博弈（真方向藏在低水） | 线位偏离标准>0.5 + 低水明确 | ⭐⭐⭐⭐ | 阿拉维斯2-4 |
| D | 深线确认 | 线位远高标准(差>0.5) + 大球低水 | ⭐⭐⭐⭐⭐ | 勒沃库森4-1 |

**模式详解**：

```
模式A：三信号共振（最强）
  0球低开(小球) + 浅线(小球) + 小球低水
  → 三重确认 → 小球方向 ✅
  案例：科莫0-0 — 0球9.75低开 + 2.25浅线 + 小0.77低水 → 0球

模式B：0球+线位共振
  0球明确方向 + 线位同向（水位可中性）
  → 双重确认 → 按0球方向走 ✅
  案例：科林蒂安2-0 — 0球7.5低开 → 小球
  案例：波特诺山丘1-1 — 0球9低开 → 小球

模式C：浅线博弈（真方向在低水）
  近况极高 + 线位被压低(浅线) + 低水在近况方向
  → 浅线 = 庄家压线制造"小球热门"假象
  → 低水方向 = 庄家真实防范方向
  案例：阿拉维斯2-4 — 近况7.4应开3.5+但只开2.25
         + 大球0.81低水 → 浅线是诱小手段 → 真实大球 ✅

模式D：深线确认
  线位远高标准 + 大球低水
  → 深线本身 = 庄家大球预期
  → 低水 = 在深线上收窄赔付
  案例：勒沃库森4-1 — 近况7.0 + 3.5深线 + 大0.91/小0.79
        深线=大球基调，低水在"小"侧只是平衡手段 → 大球 ✅
```

#### 🔥 二、假盘造热（Fake Market Heating）

**定义**：庄家刻意制造某方向的"安全感"，引导资金集中。实际反向结果。

```
特征：
  信号方向 = 庄家希望你相信的方向
  实际方向 = 信号方向的反面
```

| G | 全指向造热 | Step2.5触发5信号 | 系统推荐=造热 | 巴列卡诺1-0 |

#### 🌫️ 三、模糊地带（Ambiguous Zone）

**定义**：信号矛盾 或 都在中性区间，无法形成明确方向判断。

```
触发条件（任一即触发）：
 ① 0球在理论区间内（偏差<2）AND 线位在标准范围（偏差<0.5）
 ② 0球指向小球 BUT 线位深 + 大球低水（矛盾！）
 ③ 0球指向大球 BUT 线位浅 + 小球低水（矛盾！）
 ④ 无大小球盘口数据

处理方式：
  → 跳过方向判断，依靠后续排除法+双命中率判断
  → 置信度降低一档
  → 倾向"观望/不投注"（如果后续排除法也不明确）
```

### 0.2.5 ⚠️ 庄家异常手法识别（V3.2新增·埃尔沃斯堡+马格德堡复盘）

**核心原则：庄家开盘方向 vs 玩家预期方向 vs 庄家是否防御 = 决定真假。**
**嘴上说什么不重要，看他手上做什么。**

#### 铁律4：近况大球 + 庄家开盘跟大球 + 不防御大球 = 大球不会出

```
玩家预期：近况6.2+ → "肯定是大球！"
庄家开盘：0球23极端高开 + 线位3.25跟随 → "对大球，跟你想的一样"
庄家操作：大球区4-7球全部0次变化 → 完全不防大球！

矛盾：庄家说"大球"，但一分钱保费都不掏
→ 如果庄家真认为大球会出 → 会降赔减少赔付
→ 庄家不降赔 = 庄家不担心大球 = 大球不会出
→ 实际小球！

案例：马格德堡1-0柏林赫塔 — 近况6.2+0球23高开+全场0次→1球小球 ✅
```

#### 铁律5：近况大球 + 庄家开盘反方向引导 = 庄家在设陷阱

```
玩家预期：近况8.2+ → "肯定是大球！"
庄家开盘：线位3.25偏浅(应3.5+) + 小0.63极低水 → 引导小球 → 与玩家相反
庄家操作：0-2球全部↓降赔防范 → 加强小球假象

解读：庄家在玩家都认为大球时，反向操作引导小球
→ 极低水(<0.70) = 廉价甩卖小球 = 吸引小球资金
→ 和玩家的共识背道而驰 → 庄家在设陷阱
→ 实际大球！

案例：埃尔沃斯堡5-1帕德博恩 — 近况8.2+小0.63极低水+小球区全降→6球大球 ✅
```

#### 判断矩阵

| 庄家开盘 | 庄家是否防御 | 实际方向 | 策略 |
|---------|------------|---------|------|
| 跟玩家一致 | ✅ 有防御 | 跟随庄家 | 信庄家的手 ✅ |
| 跟玩家一致 | ❌ 无防御 | 反向 | 庄家不在乎=不会出 ✅ |
| 跟玩家相反 | — | 跟玩家 | 庄家设陷阱，玩家对 ✅ |

#### 量化的"有防御 vs 无防御"

| 防御等级 | 标志 | 判定 |
|---------|------|------|
| **强防御** | 该方向≥3个进球数↓降赔≥2次 | 实防 |
| **弱防御** | 该方向1-2个进球数↓降赔1-2次 | 弱防 |
| **无防御** | 该方向全部0次变化 | ⚠️ 异常！ |
| **反向操作** | 该方向全部↑推离 + 反向↓降赔>2次 | 🚨 陷阱 |

#### 铁律6：极低水(<0.70) = 廉价甩卖 ≠ 真实防范

```
正常低水 0.75-0.85 → 庄家防范 → 真方向
极低水 <0.70 → 白菜价甩卖 → 诱导陷阱
→ 投100只赔<70 → 玩家蜂拥 → 资金集中 → 不会打出
```

### 0.3 操作SOP

```python
# Step 0：大小球方向判断（推理第一步）V3.2

Step 1: 读四要素
  近况 = 主X + 客Y
  标准线 = 查表（近况→标准线）
  亚盘线位 = 大小球盘口线
  水位 = {大球水位, 小球水位}
  0球方向 = 0球赔率 vs 理论区间 → 低开/高开/中性
  0球偏差 = 实际0球 - 理论区间最近边界

Step 2: 判断线位方向
  if 线位 - 标准线 > +0.5: 线位方向 = "大球"
  elif 线位 - 标准线 < -0.5: 线位方向 = "小球"
  else: 线位方向 = "中性"

Step 3: 判断水位方向
  低水方 = min(大球水位, 小球水位)
  if 低水 < 0.85: 水位方向 = 低水方
  else: 水位方向 = "中性"

Step 4: 综合判定（⚠️ V3.2修正）
  信号们 = [0球方向, 线位方向, 水位方向]

  # 先检查假盘（V3.2：E/F规则大幅修正）
  # ⚠️ E规则修正：极端0球=真信号，只有"偏高但不极端+线位不跟随"才是陷阱
  if 0球偏差 > +10:
    → 信号E1：极端高开=真大球信号 → 0球方向设为"大球"（非陷阱！）
  elif 0球偏差 > +5 AND 线位 - 标准线 ≤ +0.5:
    → 信号E2：偏高0球+线位不跟随 → 可能是陷阱 → 降为模糊

  # ⚠️ F规则修正：线位压制近况
  if 线位 in [2.5, 3.0] AND 低水 < 0.85 AND 近况 ≥ 5.0 AND 线位方向 = "中性":
    → 信号F：线位压倒近况 → 低水方=真防范，不是陷阱！
    → 水位方向覆盖为"强信号"（高于0球方向）
  elif 线位 in [2.5, 3.0] AND 低水 < 0.85 AND 近况 < 5.0:
    → 信号F2：近况中性+标准线+低水 → 原F规则 → 低水方热门悖论

  # 再检查实防（V3.2：信号权重调整）
  # 信号优先级：线位方向 > 0球方向 ≈ 水位方向
  # 但当水位方向与线位方向一致时 → 水位权重提升半级
  if 信号们中 ≥2 个同向 AND 无矛盾:
    → 实盘防御（按模式A/B/C/D归类）准确率83.3%
  
  # 否则模糊
  → 模糊地带

Step 5: 输出
  方向 = "大球" / "小球" / "模糊"
  置信度 = 信号共振数量
  后续策略 = 方向明确→在方向内做排除法 | 方向模糊→全范围排除法+降置信
```

### 0.5 30场回测验证（2026-05-03新增·V3.2依据）

> 对5月2日-3日已有大小球数据的30场比赛进行V3.1 Step 0框架分析，27场有赛果可验证。

**整体结果**：

| 类型 | 场次 | 正确 | 准确率 | 说明 |
|------|------|------|--------|------|
| 🛡️ 实盘防御 | 6场 | 5场 | **83.3%** | 双信号同向=最可靠信号 |
| 🔥 假盘E(原规则) | 7场 | 2场 | 28.6% | 极端0球 ≠ 陷阱，是真信号 |
| 🔥 假盘F(原规则) | 6场 | 2场 | 33.3% | 线位压制近况，低水方真防范 |
| 🌫️ 模糊地带 | 8场 | — | — | 正确识别为不明确 |

**关键发现**：

1. **极端0球 = 真大球信号**：7场E触发中5场大球（拜仁2-3、勒沃库森4-1、霍芬海姆3-3、阿贾克斯2-2、不来梅1-3）。极端高开0球反映真实市场预期，非庄家诱导。

2. **线位压倒近况**：6场"近况>5.0 + 低水小球"中4场实际小球。当线位不跟随近况方向时，线位信号权重 > 近况信号。

3. **双信号共振 83.3%可靠**：任何两个信号（0球+线位、0球+水位、线位+水位）指向同方向且无矛盾时，跟随方向走。

## 第二步：理论盘口 vs 实际盘口

```
理论0球赔率 = 各队近况分别独立计算 + 10
  主队近况 → 主队理论0球 = 主队近况 + 10
  客队近况 → 客队理论0球 = 客队近况 + 10
  → 形成理论区间 [min(主,客)+10, max(主,客)+10]
实际0球赔率 = 数据中读取

⚠️ 注意：理论0球赔率是按两队的近况【分别独立计算】的，不是相加！
例：主近况2.6→12.6，客近况2.4→13.2 → 理论区间12.6~13.2
    而不是 (2.6+2.4)+10=15
```

| 差异 | 判定 | 含义 |
|------|------|------|
| 实际在理论区间内（偏差<2） | **正常开盘** | 庄家定价中性，认可近况 |
| 实际 > 理论区间上限+2 | **高开** | 庄家引导大球方向 |
| 实际 < 理论区间下限-2 | **低开** | 庄家引导小球方向 |

**高开案例**：主近况2.6→12.6，客近况2.4→13.2 → 理论区间12.6~13.2，实际0球=22 → 远超区间上限 → 庄家引导大球方向，需结合赔率变化判断是否真实。

## 第三步：赔率变化分析（核心！）

### 3.1 统计变化次数

```python
变化次数 = len([x for x in odds_history if x != initial_odds])
```

| 变化次数 | 判定 |
|---------|------|
| **0次** | 庄家非常自信，无需调整开盘 |
| **1-2次** | 试探性调整，或"做给玩家看" |
| **≥3次** | 庄家真实防范，持续调整 ↓ 高置信度信号 |

### 3.2 变化方向 + 水位 综合判断

```
如果 赔率下降 + 变化≥3次 + 终盘水位低
    → 庄家真实防范 → 该结果可能打出 ✅

如果 赔率下降 + 变化1-2次 + 终盘水位不低
    → 假动作/诱盘 → 该结果可能不打出 ❌

如果 赔率上升
    → 庄家不看好 → 该结果可能不打出 ❌
```

### 3.3 水位高低判断标准

| 赔率区间 | 低水位 | 中水位 | 高水位 |
|---------|--------|--------|--------|
| 0球 | < 12 | 12-18 | > 18 |
| 1球 | < 4.5 | 4.5-5.5 | > 5.5 |
| 2球 | < 3.2 | 3.2-3.8 | > 3.8 |
| 3球 | < 3.2 | 3.2-3.8 | > 3.8 |
| 4球 | < 4.5 | 4.5-5.5 | > 5.5 |
