# V4优化版算法总结

## 核心改进（基于PDF阅读）

### 1. 吸收的新概念

| 概念 | 说明 |
|------|------|
| **分布类型** | 顺分布/逆分布/中庸分布/缓冲分布 |
| **档位体系** | 人强/普强/准强/中上/中游/中下/下游 |
| **平赔调节** | 逆分布时平赔可调节性大 |

### 2. V4核心规则

#### A级（最高置信度）
1. **极低赔 + 档位差距大**：强队必胜
2. **低赔(<1.5) + 状态佳(>65)**：强队胜

#### B级（高置信度）
1. **逆分布/中庸分布 + 澳门推荐平局**：防平
2. **状态接近 + 低平赔(<3.5)**：防平
3. **顺分布 + 低赔(<1.7)**：强队胜

#### C级（一般）
- 默认跟随澳门推荐

1. **A级信号100%可靠**：极低赔+档位/状态差距大
2. **顺分布最可靠**：档位差距大时强队必胜
3. **V4优化版全面提升**：
   - 冷门日：50% → 59.4%
   - 正路日：48.3% → 69.0%
