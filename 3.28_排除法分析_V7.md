## 二、逐场V7排除法分析

### 周六001 町田泽维 vs 川崎前锋（日职）

**赔率绝对值**：竞彩初盘 1.79/3.56/3.45 → 即时 1.61/3.80/4.10，全部合理范围，无>5.0

**竞彩×澳门互动**：
- 竞彩：H↓10.1% D↑6.7% A↑18.8% → 单出口造热主胜
- 澳门：H↓16.4% D↑3.4% A↑38.4% → 同向单出口造热主胜
- **两家同向造热主胜 + 单出口 = 筹码极端聚焦主胜方向**

**造热判断**：竞彩主降10.1%（>5%）+ 澳门同向推主（降16.4%）+ 单出口 → **真造热主胜 → 排除主胜** ★★★★

**排除结果**：主胜✗ → 剩平3.80 vs 客4.10 → 赔率差0.30<0.5 → **优先考虑平局**

**辅助参考**：
- 澳门心水推町田赢，心水方向赔率≈1.61（即时）<3.0 → 心水可信度高
- 主队近况WLWWDW（6胜3平1负）远好于客队LWLDLD（2胜3平5负）
- 近况差=大正，支持主队

**矛盾**：排除法排除主胜，但澳门心水+近况都支持主胜，信号矛盾。

**结论**：★ 观望（排除法排除主胜但近况+心水支持主胜，信号矛盾；若排除主胜则防平）

- 澳门：H↑9.4% D↓2.6% A↓5.4% → 澳门推客+平，与竞彩方向相反

**排除结果**：
- [不跟主]：竞彩造热主但澳门反推 → 主胜造热假象 → **排除主胜** ★★★★
- 剩余：平2.84 vs 客2.36 → 赔率差0.48<0.5 → **优先考虑平局**

**辅助参考**：

- 近况差≈0，无方向指引
- 澳门分析："双方都缺乏争胜本钱，和味甚浓"

**结论**：★★★★ **平局**（排除主胜 + 心水推平 + 澳门分析支持 + 近况双方都差）
