## 五、框架使用方法论总结

### 决策流程图

```
收到比赛数据
    ↓
是否触发造热三维度（第零梯队）？
  ├── 是 → 造热判断流程（真/假造热→方向）
  └── 否 ↓
    
是否触发明确规律（规律一/H/O/五等）？
  ├── 是 → 规律判断
  │     ├── 友谊赛？ → 信心上限50%，可能降级
  │     ├── 竞彩矛盾？ → 规律五降级
  │     └── 无矛盾 → 明确预测
  └── 否 ↓

进入H2d框架判断
    ↓
确定澳门×竞彩互动模式
    ↓
├── ① 不怕标签 → X不打出 → 剩余中用赔率绝对值选
├── ② 澳门独控 → 以澳门方向为准
├── ③ 全不动 → 赔率绝对值最低赔付方向（需基本面配合）
├── ④ 同向 → 承接方向=可能打出
└── ⑤ 分歧 → 不参与（仅★可靠度）
    ↓
最终检查
  ├── 友谊赛？ → 信心-20%
  ├── 热门队陷阱？ → 降级
  └── 信号矛盾？ → 不参与
```

### 不参与的黄金标准

以下情况**坚决不参与**：
1. 友谊赛 + 任何规律 → 不参与（除非不怕标签★★★★★）
2. 分歧模式 ⑤ → 不参与
3. 全不动 + 近况差中性(±3) + 友谊赛 → 不参与
4. 热门队 + 分歧/信号矛盾 → 不参与
5. 规律被竞彩矛盾否定 → 不参与

### 参与的最低标准

以下情况可以**给出低信心预测**：
1. 不怕标签 → 排除后只剩1-2个方向
2. 澳门独控 → 澳门方向明确
3. 全不动 + 赔率最低方向有基本面配合
4. 同向 + 无规律矛盾
