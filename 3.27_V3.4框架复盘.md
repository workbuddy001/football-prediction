- 澳门不动 + 竞彩平升15.9%(走高) → `不怕D` → **平不打出** ★★★★★
- 澳门不动 + 竞彩客降21.6%(造热) → `不跟A` → 辅助
- `不怕H+不怕D`双重排除 → 主和平都不打出 → **排除法只剩客**？
- **但**：三重聚焦警告！澳门推客 + 竞彩客造热 + 近况客优 = 三重聚焦 = 反向信号

- 不怕标签优先级最高，但排除法导向客
- 三重聚焦是更高维度的反向信号
- **三重聚焦 > 不怕标签**（造热信号维度更高）

**V3.4结论**：**主胜**（三重聚焦反向 → 最高维度信号覆盖）

**实际**：2:0主胜 ✅

> **教训**：当三重聚焦（澳门推+造热+近况同向）与不怕标签冲突时，三重聚焦反向信号优先。因为三重聚焦是在更高维度（造热三维度）上工作的。

## 六、3.27→3.28跨日复盘的规律强化

### 新增规律/规则

1. **三重聚焦 > 不怕标签**
   - 3.27-002验证：三重聚焦（澳门推客+造热+近况同向）→ 反向主胜 ✅
   - 不怕标签排除法指向客，但三重聚焦维度更高 → 主胜
   - 3.28-002（中国vs库拉索）同样模式验证成功

2. **明确规律 > H2d不怕标签**
   - 3.27-007：规律P vs 不怕A → 规律P赢 ✅
   - 3.27-009：规律Q vs 不怕D+A → 规律Q赢 ✅
   - 规律是"明确信号"，不怕标签是"辅助排除"

3. **热门队全不动 = 信号存疑**
   - 3.27-六012：威廉二世（热门）+全不动 → 主胜预测失败 ❌
   - 3.28-六012：同一场景重复失败 ❌
   - **新增规则**：热门队 + 全不动 + 平赔极端高 → 热门队陷阱，不参与

### V3.4决策优先级（更新版）

```
第零梯队：造热三维度（最高维度）
  ├── 三重聚焦（澳门+造热+近况同向）→ 反向信号，最高优先级
  └── 其他造热组合 → 按造热流程判断

第一梯队：明确规律
  ├── 规律一（高置信度+澳门同向）
  ├── 规律O（近况差≥8+赔率<2%）
  ├── 规律Q（防过热平局）
  ├── 规律P（诱平陷阱）
  └── 规律R/N（真造热/极端反向）

第二梯队：H2d框架
  ├── 不怕标签（★★★★★排除法）
  ├── 全不动→赔率绝对值（★★★）
  ├── 澳门独控（★★★★）
  └── 同向/分歧

第三梯队：最终检查
  ├── 友谊赛+信号矛盾 → 不参与
  ├── 热门队+全不动+平赔极端 → 不参与
  └── 近况差中性+友谊赛 → 不参与
```
