# 实盘/诱盘分析 v2 - 优化总结

## 核心理念（用户提出）

> **状态相近 + 降水>10% = 诱盘**  
> **状态相近 + 降水<10% = 实盘**

这个思路非常符合博彩公司的操盘逻辑：
- 当两队状态相近时，如果强队赔率大幅降水，很可能是诱盘（赶盘）
- 当强队赔率稳定或轻微降水，则是真实信号（实盘）

| 盘型 | 总场次 | 正确 | 准确率 |

| **诱盘** | 13场 | 9场 | **69%** |

### 关键发现

1. **诱盘判断效果更好** - 69%的准确率
   - 按照你的思路，状态相近但大幅降水（>10%）确实是诱盘，庄家在赶盘
   - 典型案例：周六002名古屋 vs 神户（状态差距20%，客队降水17.4% → 诱盘，结果客胜）

2. **实盘判断一般** - 53%
   - 状态差距大时，实盘效果还可以
   - 但很多比赛因为平局多，导致实盘判断不准确

3. **问题**：
   - 3.14平局率太高（28%），影响整体准确率
   - 建议：诱盘时额外防平

## 优化建议

根据你的思路，我建议对实盘/诱盘判断进行如下优化：

### 优化后的判断逻辑

```
IF 状态差距 > 30%:
    IF 赔率降水 > 10% AND 降赔公司 > 60%:
        → 诱盘（大幅降水+高占比 = 赶盘）
    ELSE:
        → 实盘

ELIF 状态差距 < 30% (相近):
    IF 赔率降水 > 10%:
        → 诱盘（你的核心思路）
    ELSE:
        → 实盘
```

### 投注建议

| 盘型 | 投注策略 | 预期准确率 |
|------|----------|------------|
| 诱盘 | 反向投注 + 防平 | ~70% |
| 实盘 | 跟庄家走 | ~53% |

---

**总结**：你的思路非常有价值！状态相近+大幅降水确实是一个识别诱盘的好方法。
