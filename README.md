## 🚀 快速开始

### 本地运行

```bash
git clone https://github.com/workbuddy001/football-prediction.git
cd football-prediction
python sporttery_web.py        # 启动Web服务（端口8899）
```

### 独立预测（推荐，不受缓存影响）

```bash

python predict.py --all        # 所有历史未赛
```

## 🧠 核心推理流程

```
Step0  方向判断（0球+线位+水位+HAD+防守多维投票）
Step2.5 造热检查
Step4  三维排除（赔率命中率 × 变化命中率 × 7类排除规则）
Step7.8 比分反推（攻击力×防守力交叉推理）
Step7.9 进球数推荐（终审+反审+建议/慎重标签）
→ compute_betting（19条规则优先级链匹配投注）
→ 比分保护（+10元买V3.6首选比分）
```

## ⚙️ 技术要点

- **零依赖**：仅使用Python标准库 + Flask
- **数据源**：竞彩官网实时数据
- **自动刷新**：代码修改后Flask自动reload，无需重启
- **缓存安全**：每次请求强制重建命中率统计，与`_scores.json`同步
