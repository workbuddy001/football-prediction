## 五、规律V3改进建议

### 1. 规律S优化
```
原条件：|近况差|≤2 + 赔率变化>5% → 反向
优化条件：|近况差|≤2 + 赔率变化>5% + 被反向方向赔率<3.5
排除：被反向方向赔率≥3.5（赔付过高，庄家不敢造热）
```

### 2. 规律U/V优化
```
原条件：近况差≥+8 + 主胜降幅>5% + 客胜升幅>5% → 防平
优化条件：近况差≥+8 + 主胜降幅>5% + 客胜升幅>5% + 澳门推主
增加：澳门推方向与造热方向相反时才触发
```

### 3. 规律五V3优化
```
原条件：主胜升幅>5% + 客胜变化>-5% + |近况差|>3 → 平局
优化条件：主胜升幅>5% + 客胜变化>-5% + |近况差|>3 + 澳门推平
增加：澳门推平才触发规律五（澳门推主时不触发）
```
