# V3优化版算法 - 吸收《欧赔核心思维》核心思维

## 一、核心概念（从PDF提取）

### 1. 人气指数体系
- **一档球队**：切尔西、巴萨、尤文、皇马、AC、曼联、阿森纳
- **准一档**：国米、拜仁、利物浦、里昂
- **二档球队**：维拉、贝迪斯、沙尔克、波图、PSV、阿贾克斯、不来梅、乌迪内
- **三档/四档**：更低级别球队

### 2. 即时广义实力差因素
- 人气形势（如刚大胜后人气上升）
- 渊源（如德比战）
- 战意（保级/争冠）
- 主客场战绩
- 往绩（胜负多/交叉/平局多影响不同）
- 核心球员伤停
- 联赛特性
- 排名
- 比赛时段

### 3. 欧赔核心招式

#### 平赔分散思维
- **拉低平分散胜赔**：平赔下降但不过分，分散胜赔投注
- **抬胜分平**：抬高血压分散平局
- **平赔过渡均衡**：平赔保持在心理区间

#### 分散投注分布
一场比赛庄家需要将投注分散到三个方向，否则会承受风险。

### 4. 关键信号

#### 真盘思维
- 初盘即终盘，核心公司赔率不变 = 强烈信号

#### 诱导分散
- 立博反向操作（如主胜大幅下降但客胜未动）= 诱导分散

#### 极限诱导
- 超低主胜赔率 + 低平赔 = 陷阱盘

#### 凯利值
- 某项凯利值超过100% = 庄家在保护该位置

## 二、V3优化版判断规则

### A级（最高置信度）：同时满足多个强信号

1. **真盘思维 + 低赔集中**
   - 核心公司（威廉/立博/竞官方）赔率不变
   - 主胜赔率在1.5以下强区间
   - 预测：主胜

2. **极限诱导识别**
   - 主胜超低(<1.35) + 平赔极低(<3.0) = 防平
   - 但主胜超低同时客胜上升 = 主胜（庄家不怕）

3. **状态差距极大 + 低赔实盘**
   - 主队状态胜率>60%，客队<30%
   - 主胜赔率<1.5
   - 预测：主胜

### B级（高置信度）

1. **拉低平分散胜赔**
   - 主胜赔率偏高(1.6-2.0)
   - 平赔小幅下降(5-10%)
   - 澳门推荐主胜
   - 预测：主胜

2. **抬胜分平**
   - 主胜抬升
   - 平赔上升但保持梯度
   - 客胜偏低
   - 预测：防平

3. **人气差距 + 合理低赔**
   - 强队人气指数高
   - 赔率在合理区间（1.5-1.7）
   - 预测：主胜

### C级（一般置信度）

1. **状态相近 + 平赔稳定**
   - 主客胜率接近
   - 赔率变化小(<5%)
   - 跟随主流公司

2. **澳门推荐平局 + 平赔下降**
   - 澳门推荐平局
   - 平赔下降>10%
   - 预测：防平

3. **强队被看衰**
   - 强队赔率上升>15%
   - 可能诱盘，预测防冷

## 三、实战应用要点

1. **先判断人气指数**：明确双方在人气体系中的位置
2. **分析即时广义实力差**：状态、战意、往绩等
3. **观察核心公司赔率**：威廉、立博、竞官方
4. **识别做盘思维**：拉低平？抬胜分平？诱导分散？
5. **计算凯利值**：判断庄家保护方向

### 问题分析
- 3.13平局漏判严重（实际5场平局）
- 需要加强平局识别

### 改进方向
1. 当平局比例异常高时，增加防平权重
2. 澳门推荐平局时应更果断
3. 关注"抬胜分平"思维
