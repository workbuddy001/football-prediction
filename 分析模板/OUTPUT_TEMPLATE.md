### 离散度格式说明

- 格式：`H-10.9 D+2.3 A+13.1 → H-4.2 D+5.6 A-2.8`
- H/D/A = 竞彩赔率相对澳门赔率的差值百分比（以澳门为基准）
- 正数 = 竞彩赔率高于澳门，负数 = 竞彩赔率低于澳门
- 前半段 = 初盘离散度，后半段 = 即时离散度
- 差值绝对值越小 → 两家观点越一致

### 一致性判断标准

| 离散度均值 | 一致性 | 含义 |
|:----------:|:------:|------|
| < 2% | 高度一致 | 两家赔率几乎相同，观点完全统一 |
| 2% ~ 5% | 基本一致 | 存在微小差异，但方向一致 |
| 5% ~ 10% | 轻度分歧 | 有一定差异，需注意 |
| 10% ~ 15% | 明显分歧 | 两家观点有显著不同 |
| > 15% | 严重分歧 | 两家观点严重冲突，庄家在操控（竞彩偏低方可能被庄家引导） |

> **核心用途**：离散度小 = 两家观点一致 → 信号可信度高。离散度大 = 两家分歧大 → 需谨慎判断，可能存在庄家操控。
> 竞彩赔率通常低于澳门（竞彩有返奖率限制），主胜和客胜方向的离散度差异可以反映庄家引导方向。

### 变化(H/D/A)格式说明

- 格式：`H+13.2% D-14.1% A-18.3%`
- H：主胜变化百分比，正数表示升，负数表示降
- D：平局变化百分比
- A：客胜变化百分比
- 不变用 `—` 表示

### 竞彩vs澳门 对比栏说明

> **核心用途**：竞彩和澳门赔率变化的**方向对比**，作为造热判断的辅助维度。

| 对比结果 | 含义 | 判断参考 |
|----------|------|----------|
| **同向** | 竞彩和澳门赔率变化方向一致（都升或都降） | 信号可信度高，造热判断更确定 |
| **分歧** | 竞彩和澳门赔率变化方向相反 | 信号矛盾，需谨慎判断造热 |
| **澳门不动** | 澳门赔率不变(全0%)，竞彩有变化 | 澳门静观，竞彩更敏感；造热信号以竞彩为准但需降低信心 |
| **竞彩不动** | 竞彩赔率不变(全0%)，澳门有变化 | 较少见；以澳门为准 |

**判断逻辑**：
1. **同向**：两家信号一致 → 造热/分流判断可信度 **高**
2. **分歧**：方向相反 → 需看变化幅度：
   - 竞彩变化幅度 >> 澳门变化幅度 → 竞彩更敏感，以竞彩方向为参考
   - 澳门变化幅度 >> 竞彩变化幅度 → 澳门更精准，以澳门方向为参考
3. **澳门不动**：澳门静观=庄家无引导信号 → 竞彩的变化可能是市场自然波动而非庄家操控 → 造热信号**弱化**
4. **竞彩不动**：竞彩没反应但澳门在调 → 竞彩滞后，以澳门为准

**典型场景**：
- 澳门不动+竞彩单方向大幅变化 → 竞彩市场自发造热，但庄家（澳门）未参与 → **假造热可能**（市场情绪驱动，非庄家操控）
- 两家同向大幅变化 → 庄家+市场共振 → **真造热信号强**
- 两家方向完全相反 → 信号矛盾 → **观望**

### 🎯 澳门盘口硬度分析（核心信号）

> **核心原理**：澳门是庄家操控的核心盘口。当竞彩赔率大幅变化而澳门纹丝不动时，说明澳门盘口非常"硬"——庄家对当前赔率很有信心，不随市场波动调整。

#### 场景一：竞彩大幅变化 + 澳门不动 → 澳门盘口硬

| 条件 | 判断 | 含义 |
|------|------|------|
| 竞彩变化总和 > 10% + 澳门全0% | ⚠️ **澳门盘口硬** | 庄家很笃定，不惧市场波动 |
| 竞彩单方向变化 > 5% + 澳门全0% | ⚠️ **澳门盘口硬** | 庄家对该方向无调整意愿 |

**核心推论**：
- 澳门不动 = 庄家认为当前赔率是合理的，无需调整
- 竞彩的变化反映的是**公众情绪/市场自发行为**，不是庄家引导
- 此时**以澳门的即时赔率为准**，竞彩的变化方向仅作为参考

#### 场景二：竞彩往高水走 + 澳门不动 → 澳门不怕该选项打出

| 条件 | 判断 | 含义 |
|------|------|------|
| 竞彩某方向赔率**升**(高水) + 澳门该方向**不动** | 🟢 **澳门不怕该方向打出** | 庄家不提高该方向赔率=不怕赔付 |
| 竞彩某方向赔率**降**(低水) + 澳门该方向**不动** | 🟡 **澳门不跟随造热** | 庄家不参与竞彩的降温操作 |

**详细解读**：
- 竞彩主赔升高 + 澳门主赔不动 → 竞彩认为主队更难赢了，但澳门不怕主队赢 → **澳门看好主队**
- 竞彩客赔升高 + 澳门客赔不动 → 竞彩认为客队更难赢了，但澳门不怕客队赢 → **澳门看好客队**
- 竞彩平赔升高 + 澳门平赔不动 → 竞彩认为平局更少了，但澳门不怕平局 → **澳门不排斥平局**

#### 场景三：竞彩往低水走 + 澳门不动 → 反向警惕

| 条件 | 判断 | 含义 |
|------|------|------|
| 竞彩某方向赔率**降**(低水) + 澳门该方向**不动** | 🔴 **庄家不跟随降温** | 竞彩在引导筹码去该方向，澳门不参与=不希望筹码去那里 |
| 竞彩某方向大降(>5%) + 澳门不动 + 澳门推该方向 | ⚠️ **竞彩造热，澳门不配合** | 澳门推的方向竞彩在造热，但澳门自身不跟 → 信号矛盾 |

**综合判断规则**：

```
澳门不动时：
  ├── 竞彩升(高水)的方向 → 澳门不怕 → 该方向可能有料
  ├── 竞彩降(低水)的方向 → 澳门不跟 → 竞彩在引导，庄家不配合 → 警惕被引导
  └── 结论：看澳门即时赔率绝对值，按正常规律分析（忽略竞彩变化方向）
```

#### 标注格式

在分析报告中，对"澳门不动"场次增加标注：

| 标注 | 格式 | 示例 |
|------|------|------|
| 盘口硬 | `澳门不动[硬]` | 方向对比列显示 `澳门不动[硬]` |
| 不怕高水方向 | `不怕H` / `不怕A` / `不怕D` | 竞彩主赔升+澳门不动 → `澳门不动[不怕H]` |
| 不跟低水方向 | `不跟H` / `不跟A` / `不跟D` | 竞彩客赔降+澳门不动 → `澳门不动[不跟A]` |

**判断优先级**：
1. 澳门不动 + 竞彩变化小(<3%) → 正常，无特殊信号
2. 澳门不动 + 竞彩某方向升>5% → **澳门不怕该方向** → 该方向值得关注
3. 澳门不动 + 竞彩某方向降>5% → **澳门不跟降温** → 竞彩在引导，警惕
4. 澳门不动 + 竞彩多方向大幅变化(>10%) → **盘口极硬** → 完全以澳门为准

## 离散度变化趋势

| 趋势 | 含义 | 判断参考 |
|------|------|----------|
| **缩小** | 两家赔率靠拢 | 市场共识增强，信号更可信 |
| **扩大** | 两家赔率背离 | 庄家可能在调整策略或操控 |
| **不变** | 离散度稳定 | 两家维持原有分歧 |

## 近况评分计算方法

- **权重**：最近一场×2，其他4场×1（共6场权重）
- **得分**：赢=3分，平=1分，输=0分
- **满分**：3×2 + 3×4 = 18分
- **近况差** = 主队得分 - 客队得分
