## 四、使用说明

### 步骤1: 填写比赛信息
在"比赛基本信息"部分填写：
- 主队、客队、比赛时间、赛事
- 近况走势(W/L/D)、盘路走势
- 历史交锋战绩
- 澳门心水推荐

### 步骤2: 填写赔率数据
- 初盘赔率：比赛开始前的赔率
- 即时赔率：临场最新赔率
- 顺序必须一一对应

### 步骤3: 复制到分析工具
将填写好的数据复制到 `赔率分析工具.py` 的对应位置：

```python
if __name__ == "__main__":
    
    # 比赛信息
    home_team = "维罗纳"
    away_team = "热那亚"
    match_time = "2026-03-15 19:30"
    league = "25/26意甲第29轮"
    home_form = "WLLLDL"
    away_form = "WLWDLL"
    home_handicap = "W1/LLLDL"
    away_handicap = "WLW1/LLL"
    history = "维罗纳 2胜4和4负"
    macao_tip = "和局"
    
    # 赔率数据
    initial_odds = [
        (2.88, 2.80, 2.35),
        ...
    ]
    
    realtime_odds = [
        (3.30, 2.69, 2.20),
        ...
    ]
```

### 步骤4: 运行分析
```bash
python 分析模板/赔率分析工具.py
```

## 五、注意事项

1. **赔率顺序必须一致**：初盘和即时赔率的每家公司顺序必须完全对应
2. **公司数量建议**：至少需要20家以上才能保证分析准确性
3. **澳门位置**：澳门通常是第3家公司(索引为2)
4. **数据来源**：建议从可靠渠道获取原始赔率数据
