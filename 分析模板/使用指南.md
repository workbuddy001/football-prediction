# V5 综合事件分析框架 - 使用指南

## 快速开始

### 方式1：分析单场比赛（推荐）

```python
# 导入分析器
from v7_8_segment_analyze_v3 import ComprehensiveEventAnalyzer

# 创建分析器
analyzer = ComprehensiveEventAnalyzer()

# 准备比赛数据
match_data = {
    'home_team': '里斯本',
    'away_team': '博德闪耀',
    'league': '欧冠',
    'confidence': 59,              # 置信度
    'confidence_option': '主胜',    # 置信度选项
    'form_diff': -30,              # 状态差
    'home_8': -2, 'draw_8': 2, 'away_8': 1,  # 8变化
    'home_form': 'WDWDLW',         # 主队近况（WDL字符串）
    'away_form': 'WWWDWW',         # 客队近况
    'first_leg_result': '0-3惨败', # 首回合结果（淘汰赛）
}

# 执行分析
result = analyzer.analyze_moderate_distribution(match_data, verbose=True)

# 查看结果
print(f"预测: {result['prediction']}")
print(f"方法: {result['method']}")
print(f"理由: {result['reason']}")
```

### 方式2：批量分析多场比赛

```python
from 回溯分析_V3完整框架 import MATCH_DATABASE
from v7_8_segment_analyze_v3 import ComprehensiveEventAnalyzer

analyzer = ComprehensiveEventAnalyzer()

for date, matches in MATCH_DATABASE.items():

        # 构建数据

            'away_team': match['away'],
            'league': match['league'],

            'confidence_option': match['conf_option'],
            'form_diff': match['form_diff'],

        # 分析

```

## 数据准备

### 必需字段

| 字段 | 说明 | 示例 |
|------|------|------|
| `home_team` | 主队名称 | '里斯本' |
| `away_team` | 客队名称 | '博德闪耀' |
| `league` | 赛事类型 | '欧冠' |
| `confidence` | 置信度 | 59 |
| `confidence_option` | 置信度选项 | '主胜'/'平局'/'客胜' |
| `form_diff` | 状态差 | -30 |
| `home_8` | 主胜8变化 | -2 |
| `draw_8` | 平局8变化 | 2 |

### 可选字段（用于自动提取事件）

| 字段 | 说明 | 示例 | 提取的事件 |
|------|------|------|-----------|
| `home_form` | 主队近况 | 'WWDLWW' | 近期3连胜/连败 |
| `away_form` | 客队近况 | 'LLLDLL' | 近期3连胜/连败 |
| `first_leg_result` | 首回合结果 | '0-3惨败' | 首回合惨败/失利 |
| `h2h_record` | 交锋记录 | '2胜1平2负' | 交锋优势/劣势 |
| `injuries` | 伤停列表 | ['核心前锋'] | 核心伤停 |

## 事件类型速查表

### 自动识别的事件

系统会自动从数据中识别以下事件：

**1. 近期战绩（从WDL字符串）**
- `近期3连胜` - 最近3场都是W
- `近期2连胜` - 最近2场都是W
- `近期3连败` - 最近3场都是L
- `近期2连败` - 最近2场都是L

**2. 首回合结果（从first_leg_result）**
- `首回合惨败` - 包含"0-3", "0-4", "1-4"等
- `首回合大败` - 包含"0-2", "1-3"等
- `首回合失利` - 包含"负"
- `首回合不胜` - 包含"平"

### 手动输入的事件

对于无法自动提取的信息，可以直接在`events`字段中指定：

```python
match_data = {
    # ... 其他字段
    'events': ['核心伤停', '交锋劣势明显'],
}
```

result = {

    'prediction': '主胜',                     # 预测结果

### 判断方法说明

| 方法 | 含义 | 操作建议 |
|------|------|----------|
| `实盘` | 跟随赔率置信度 | 相信强队 |
| `中庸陷阱` | 状态与分布不匹配，可能诱盘 | 谨慎防平 |
| `中庸陷阱-弱` | 轻微不匹配，需谨慎 | 可考虑防平 |

## 常见问题

### Q1: 如何获取8变化数据？

从比赛数据文件中提取：
```python
from v7_8_segment_analyze import calculate_8_change

# 提供初盘和即时赔率
init_odds = [(2.88, 2.80, 2.35), ...]  # 初盘
real_odds = [(2.75, 2.85, 2.45), ...]  # 即时

eight_change = calculate_8_change(init_odds, real_odds)
print(eight_change)
# {'home_8': -2, 'draw_8': 2, 'away_8': 1, ...}
```

### Q2: 如何获取状态差？

从胜率计算：
```python
# 主队胜率60%，客队胜率30%
form_diff = 60 - 30  # = 30
```

### Q3: 首回合结果怎么填？

格式建议：
- `'0-3惨败'` - 输3球+
- `'0-2大败'` - 输2球
- `'0-1负'` - 输1球
- `'1-1平'` - 平局
- `'2-0胜'` - 赢2球
- `'3-0大胜'` - 赢3球+

### Q4: 没有首回合的比赛怎么办？

不填`first_leg_result`字段即可，系统会自动识别为联赛/无首回合。

## 高级用法

### 自定义事件权重

```python
analyzer = ComprehensiveEventAnalyzer()

# 修改权重
analyzer.event_weights['首回合惨败'] = -0.50  # 原来是-0.40
analyzer.event_weights['近期3连败'] = -0.35   # 原来是-0.30

# 添加新事件
analyzer.event_weights['新事件'] = -0.20
```

### 批量处理并导出结果

```python
import json

results = []
for match in matches:
    result = analyzer.analyze_moderate_distribution(match)
    results.append({
        'match': result['match'],
        'prediction': result['prediction'],
        'method': result['method'],
        'reason': result['reason'],
    })

# 导出为JSON
with open('预测结果.json', 'w', encoding='utf-8') as f:
    json.dump(results, f, ensure_ascii=False, indent=2)
```

## 总结

**三步使用法：**
1. 准备比赛数据（必需字段 + 可选字段）
2. 调用`analyze_moderate_distribution()`
3. 查看`prediction`、`method`、`reason`

**核心原则：**
- 有负面事件 → 市场预期已降 → 中庸合理 → 跟随实盘
- 无负面事件 → 市场预期正常 → 中庸异常 → 可能诱盘
