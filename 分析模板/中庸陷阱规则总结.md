# 中庸分布与状态不匹配规则总结

## 核心思路
**中庸分布应该符合两队状态中庸**

当一支球队状态极好，但赔率变化呈中庸分布时，说明市场没有明显倾向，这可能是：
1. 庄家真实看好但不引导（强队仍能赢）
2. 诱盘陷阱（需要防平）

## 规则演变

### V1（初始版本）
- 状态极好（>=30%）+ 中庸分布 → 防平
- **问题**：过于敏感，强队即使中庸分布也能赢

### V2（3.17验证后优化版）
**保护规则：**
1. 置信度>=70%：不触发陷阱（强队保护）
2. 状态差>=40%：不触发陷阱（状态极好保护）

**触发条件：**
- 状态差30-40% + 平局8变化>=4 → 防平
- 状态差20-30% + 平局8变化>=5 → 防平

### V3（3.17欧冠教训优化版）
**关键发现：**
这两场都是欧冠淘汰赛**次回合**：
- 里斯本首回合客场 0-3 惨败给博德闪耀
- 阿森纳首回合客场 1-1 被勒沃库森逼平

**核心洞察：**
首回合客场不利 + 次回合中庸分布 = **庄家真实看好主队反弹**，不是陷阱！

**新增保护规则：**
3. 首回合客场不胜/惨败 + 次回合中庸分布：不触发陷阱（反弹信号）

### V4（近期关键事件框架）
**核心升级：**
引入"近期关键事件分析"，判断市场预期是否已经调整

**事件权重体系：**
| 事件类型 | 权重 | 说明 |

| 首回合惨败 | -0.35 | 市场信心大降 |
| 首回合失利 | -0.25 | 市场信心下降 |
| 首回合不胜 | -0.20 | 市场信心略降 |
| 近期连败 | -0.25 | 状态低迷 |

**新判断逻辑：**
- 事件调整值 <= -0.25：市场预期已调整，中庸分布合理，**不触发陷阱**
- 事件调整值 >= -0.1：市场预期正常，中庸分布异常，**可能触发陷阱**

**四层判断逻辑：**
```

第二层：状态保护（状态差>=40%）→ 不触发
第三层：市场预期判断（事件调整值）
    - <= -0.25：市场预期已降 → 中庸合理 → 不触发
    - >= +0.25：市场预期已升 → 可能诱盘 → 进一步判断
    - -0.25~0.25：市场预期正常 → 中庸异常 → 可能触发
第四层：中庸陷阱触发条件
    - 状态好(30-40%) + 平8>=4 + 无负面事件 → 触发
    - 状态较好(20-30%) + 平8>=5 + 无负面事件 → 触发
```

## 当前规则逻辑（V5 - 综合事件框架）

```python
class ComprehensiveEventAnalyzer:
    def __init__(self):
        self.event_weights = {
            # 首回合结果
            '首回合惨败': -0.40, '首回合大败': -0.30, 
            '首回合失利': -0.25, '首回合不胜': -0.15,
            '首回合小胜': +0.10, '首回合中胜': +0.20, '首回合大胜': +0.35,
            # 近期战绩
            '近期3连败': -0.30, '近期2连败': -0.20,
            '近期3连胜': +0.30, '近期2连胜': +0.20,
            # 交锋记录
            '交锋3连败': -0.25, '交锋3连胜': +0.25,
            # 伤停
            '核心伤停': -0.25, '多主力伤停': -0.35,
            # 主客场
            '主场龙': +0.15, '主场虫': -0.15,
        }
    
    def analyze_moderate_distribution(self, match_data):
        """分析中庸分布的含义"""
        # 提取数据
        conf = match_data['confidence']
        form_diff = match_data['form_diff']
        home_8 = match_data['home_8']
        draw_8 = match_data['draw_8']
        away_8 = match_data['away_8']
        
        # 自动提取事件
        events = self.extract_events_from_data(match_data)
        
        # 计算事件调整值
        adjustment = sum(self.event_weights.get(e, 0) for e in events)
        
        # 判断是否为中庸分布
        is_moderate = abs(home_8) <= 2 and abs(draw_8) <= 2 and abs(away_8) <= 2
        
        if not is_moderate:
            return {'prediction': conf_option, 'method': '实盘', 'is_trap': False}
        
        # === 四层判断逻辑 ===
        
        # 第一层：强队保护
        if conf >= 70:
            return {'prediction': conf_option, 'method': '实盘', 
                   'reason': f'强队保护({conf}%)', 'is_trap': False}
        
        # 第二层：状态保护
        if abs(form_diff) >= 40:
            return {'prediction': conf_option, 'method': '实盘',
                   'reason': f'状态极好保护({form_diff}%)', 'is_trap': False}
        
        # 第三层：市场预期判断
        if adjustment <= -0.25:
            return {'prediction': conf_option, 'method': '实盘',
                   'reason': f'市场预期已调整({adjustment:+.2f})', 'is_trap': False}
        
        if adjustment >= 0.25:
            # 市场预期已提升，如果还中庸，可能是诱盘
            if form_diff >= 30 and draw_8 >= 3:
                return {'prediction': '平局', 'method': '中庸陷阱',
                       'reason': '市场预期已提升但中庸，可能诱平', 'is_trap': True}
        
        # 第四层：中庸陷阱触发条件
        if -0.25 < adjustment < 0.25:
            if 30 <= form_diff < 40 and draw_8 >= 4:
                return {'prediction': '平局', 'method': '中庸陷阱',
                       'reason': f'状态好({form_diff}%)但中庸+平8大+无负面事件', 'is_trap': True}
            if 20 <= form_diff < 30 and draw_8 >= 5:
                return {'prediction': '平局', 'method': '中庸陷阱',
                       'reason': f'状态较好({form_diff}%)但中庸+平8大+无负面事件', 'is_trap': True}
        
        # 默认跟随实盘
        return {'prediction': conf_option, 'method': '实盘',
               'reason': '跟随实盘', 'is_trap': False}
```

## 使用建议

1. **高置信度比赛（>=70%）**：即使中庸分布也优先相信强队
2. **中等置信度（55-70%）+ 状态好（30-40%）+ 平局8变化大**：考虑防平
3. **有近期负面事件**：市场预期已调整，中庸分布合理，不防平
4. **无近期负面事件**：市场预期未调整，中庸分布异常，可能诱盘
5. **低置信度（<55%）**：不纳入分析

## 关键洞察

### 中庸分布的两种解读

| 场景 | 解读 | 操作建议 |
|------|------|----------|
| 无负面事件 + 状态好 + 中庸 | 市场预期未调整，可能是诱盘 | 防平 |
| 有负面事件 + 状态好 + 中庸 | 市场预期已调整，中庸合理 | 跟随强队 |

### 核心原则

**你的核心思路是对的：**
> "赔率也是顺着最近的比赛状态变化而变化，符合实盘"

**关键理解：**
- **为什么中庸** 比 **是否中庸** 更重要
- 有负面事件 → 市场信心已降 → 中庸分布是合理的
- 无负面事件 → 市场信心正常 → 中庸分布可能是诱盘
