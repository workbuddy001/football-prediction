# 2026-04-21 足球预测

## 高命中率规律发现（进球数双选）

基于221场比赛回测，发现以下高命中率规律：

### 核心规律
| 规律 | 条件 | 命中率 | 样本 |
|------|------|--------|------|
| **A** | 1+2组合 + 最低赔率<3.5 + 近况<2.5 | **100%** | 6场 |
| **B** | 近况<2.0 + 2+3组合 | **81.8%** | 11场 |
| **C** | 1+2组合 + 最低赔率<3.5 | **76.5%** | 17场 |
| **D** | 1+2组合全局 | **68.2%** | 22场 |

### 关键发现
- **1+2组合是最佳双选**，当赔率接近（比值1.1-1.3）时命中率75%
- **近况弱（<2.0）时，2+3组合有81.8%命中率**
- **赔率差距大（>1.3）时危险，命中率仅25%**

### 回测验证
- 高置信推荐: 28场 命中22场 命中率 **78.6%**
- 覆盖率: 33场 / 221场 = 14.9%
