## 新增：赔率离散度维度

### 离散度计算
- 公式：(竞彩赔率 - 澳门赔率) / 澳门赔率 × 100%
- 分别计算初盘和即时两个时间点的离散度
- 三向(H/D/A)取绝对值平均为总体离散度

### 一致性判断
- <2% 高度一致 / 2-5% 基本一致 / 5-10% 轻度分歧 / 10-15% 明显分歧 / >15% 严重分歧

## 新增：澳门盘口硬度分析（H2c规则）

### 核心洞察
- 竞彩大幅变化 + 澳门不动 = 澳门盘口很"硬"，庄家很笃定
- 竞彩往高水走 + 澳门不动 = 澳门不怕该方向打出（该方向可能有料）
- 竞彩往低水走 + 澳门不动 = 庄家不跟降温（竞彩在引导，庄家不配合）

### 标注格式
- `澳门不动[硬]` — 竞彩变化>10%，盘口极硬
- `澳门不动[不怕H/A/D]` — 竞彩某方向升>5%，澳门不怕
- `澳门不动[不跟H/A/D]` — 竞彩某方向降>5%，澳门不跟
