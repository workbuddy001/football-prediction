## 踩坑/教训（续）
- 两处路由处理（POST和GET）都要加新的API端点，否则一种请求方式不工作
- replace_in_file遇到重复old_string时需要加更多上下文使其唯一
- **弹窗显示/隐藏必须统一**：classList.add('show') vs style.display='flex' 混用导致 closeDetail() 关不掉 → 统一用 detailContent + classList
- **不能覆盖外层容器detailBox**：所有弹窗内容应写入 detailContent 子元素，覆盖外层会导致子元素丢失→后续报错 Cannot set properties of null
- **冷门相似度阈值调优**：后端>=20 + 前端>=20 才显示；三级警告：>=50危险红 / >=35警示黄 / 20~29参考蓝
- **端口是8899不是5000也不是9000**：代码默认8899，用户之前可能手动指定过9000
