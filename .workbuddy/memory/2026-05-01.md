## 2:2高命中率比分 vs 大球统计验证

**用户假设**：当历史高命中率比分含2:2时，总进球>3球概率很高。

**验证脚本**：`_backtest_22_hitrate.py`

**方法**：按match_id时间顺序逐场回测，用历史数据（仅当前场之前的比赛）计算各比分在各赔率区间的命中率，命中率>=9%且样本>=5场视为"高命中率"。

**数据量**：423场有效比赛（2038719-2039415），跳过前20场建库。

**结果**：❌ 假设不成立

| 分组 | 场数 | 大球(>3) | 大球率 |
|------|------|----------|--------|
| A: 2:2高命中率 | 65 | 20 | 30.8% |
| B: 2:2非高命中率 | 358 | 109 | 30.4% |
| C: 2:2未开出 | 0 | 0 | - |

- A vs B 差异仅 +0.4%，倍率1.01x，无统计差异
- 2:2高命中率时进球分布：0球3场 / 1球9场 / 2球17场 / 3球16场 / 4球10场 / 5球8场 / 7球1场 / 9球1场

**细化分析**（按2:2命中率档次）：
- >=9%(高): 30.8% | 5-9%(中): 31.3% | 1-5%(低): 23.8% | 0%(无历史): 42.9%
- 各档大球率均在30%左右，无单调趋势

**结论**：2:2出现在高命中率比分中，不具大球预测价值。该假设被统计证伪。

## 2:1/1:2高命中率 vs 3球统计验证

**用户问题**：历史高命中率比分含2:1或1:2时，恰好3球打出概率？

**验证脚本**：`_analyze_21_12_3goal.py`

**结果**：❌ 含2:1/1:2高命中率时3球率仅23.1%（69/299），低于全样本基准25.6%，远低于不含高命中率的对比组32.7%。是负信号。

**特征对比（打出3球69场 vs 未打出230场）**：
- 打出组主近况更高(2.91 vs 2.72)，HAD主胜赔上升(推离主胜)时更易3球
- 打出组客胜赔下降(造热客胜)时3球率更低(-7.6% vs +2.8%)
- 0球赔率越低越易3球(15.61 vs 16.10)
- **亮点组合**：0球10-12 + 近况3.0-3.5 → 44.4% 3球率（9场中4场）
- HAD平赔>=3.5时3球率降至20%，<3.5时约25%
