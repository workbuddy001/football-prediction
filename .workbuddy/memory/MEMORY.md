# 足球预测项目长期记忆

## 数据文件位置
- **比分结果**：`分析模板/_scores.json`（key: match_id 或 "4.17_周五001"）
- **比分字段**：`home_score` + `away_score`
- **竞彩赔率**：`sporttery_data/*.json`（每match_id一个文件）
- **分析模板**：`分析模板/YYYY.MM.DD/编号_主队vs客队_源数据.md`
- **跳过**：无match_id或match_info为空的sporttery_data文件

## 近况数据
- **来源**：`preview.recent`（dict，非list！）→ `recent.get('home',{}).get('matchList',[])`
- **得分字段**：`homeTeamFullCourtGoalCnt` / `awayTeamFullCourtGoalCnt`（不要解析fullCourtGoal字符串）
- **判断主客队**：`homeTeamShortName == teamShortName` → 主队；否则客队
- **近况规则**：<2.0→-8 | 2.0~2.5→-3 | 2.5~3.5→+3 | 3.5~4.0→-3 | >4.0→-8

## 术语定义
- **竞彩**=500.com"竞彩官方" | **澳门**=500.com"澳门"
- 源数据：第二节=初盘（idx 0=竞彩, 2=澳门），第三节=即时盘
- ⚠️ initial_odds/realtime_odds第一行是注释，实际idx 0从第二行开始
- ⚠️ 前端JSON解析：`typeof x==='string'?JSON.parse(x):x`，不要对res.raw_data再JSON.parse
- ⚠️ realtime_odds从ri=0开始遍历（不是ri=1）
- ⚠️ 竞彩数据`+1`=主队受让，`-1`=主队让球

## 核心规律速查

### 3球规律
- **黄金3球**（4条件同时满足）：评分≥15 + 近况2.5~3.5 + 3球3.0~3.50 + 2球>3球<4球
- **超级3球**：黄金3球+0球=13 → 60-75%命中率
- **3球3.2-3.4是最强信号**：不管大球涨还是降，都有显著提升
- **三条件排除3球**：近况2.5~3.5 + 0球>13 + 当前3球>3.50 + 初始3球>3.50

### 3球通用规律（2026-05-01新增，423场全量回测）
- **最强命中**：让负1.50-1.70 + 主让-1 + 3球赔率3.3-3.5 → **55.6%**(18场)
- **通用命中**：让负1.50-1.70 + 主让-1 → **35.7%**(70场) ← 不依赖历史高命中率
- **通用命中**：让负1.50-1.70 + 0球8-15 → **39.5%**(43场)
- **强排除**：客近况<2.0 → **6.2%**(16场)
- **强排除**：HAD平赔降>3% → **9.1%**(11场)
- **强排除**：0球>=15 + 客近况<2.5 → **11.1%**(18场)
- **强排除**：主近况<2.0 → **12.5%**(24场)
- **含2:1/1:2高命中率时增强**：0球10-12+客近况3.0-3.5 → **50.0%**(10场)
- 已接入前端（get_final_recommendation + sporttery_web.py HTML）

### 大3球预判
- 大球涨+近况<3.5+3球3.2-3.4 = 64.3%
- 大球降+3球3.2-3.4+近况≥3.5 = 66.7%
- 0球<10+高球下降>-5% = 19.2%（vs对照组6.7%，2.9倍）

### 半全场辅助判断（5.19节，2026-04-28新增）
- **触发**：对全场方向不确定 + 有hafu_change数据
- **核心**：半全场拆出3条通向同一结果的路径（如平局=平平/负平/胜平）
- **3条路径全降** = 强信号，强度>全场单一变化3倍
- **平平降>5%** = 庄家怕"半场就平" → 可能小球平
- **负平+胜平降 > 平平不变** = 庄家怕"追平" → 可能大球平(2-2/3-3)
- **案例**：巴列卡诺3-3皇家社会（#2039343）— 平平↓10%+负平↓7.1%+胜平↓7.1% → 全场平↓5.5%双重验证

### 让球规律（341场回测）
- 主让1球→推荐让负(48.2%) | 主受让1球→推荐让胜(57.7%)
- 让胜<1.60→让胜(57.1%) | 让负<1.60→让负(59.2%)
- 中赔(3.65~3.95)+主受让+客近况好+让胜赔更低→让胜88.2%

### 推理规则优先级
- 5.8节（极端情况预警）> 5.12节（赔率跳跃诱导）
- 5.18节（全场0次变化+小球数据）> 5.14节（全场0次变化决策树）
- 弱推离(<5%)不自动排除；最低赔方向被全市场造热=诱导

## 关键教训（精简版）
1. 友谊赛全面降级，排除1个后应观望
2. 平局是最大盲区，排除1个方向后不能默认选另一方向
3. [不怕X]赔率<3.5不可靠
4. 心水推"和局"且≥3.5时不能排除平局
5. 单出口+全面造热=造热陷阱（圣旺红星验证）
6. 心水推X+澳门造热X=烟雾弹，排除X
7. 竞彩几乎不动=实盘
8. 掩护模式：筹码被吸引去某方向→另一方向安全打出
9. 近况是弱信号，近况极低(<3.5)不能默认小比分
10. 全场0次变化+数据全指小球=庄家隐藏大球方向
11. 变化命中率（样本充足）> 赔率命中率（样本不足）（波特诺山丘1-1复盘）
12. 变化命中率（样本充足≥10）> 系统排除规则（小样本）（科林蒂安2-0复盘）
   → 系统排除规则如"X<3.3+Y≥6.5+主让-1→8.3%"样本通常很小
   → 当变化命中率≥15%且样本≥10场时，应推翻小样本的系统排除规则
