## 澳门心水×竞彩赔率变化 完整信号体系（用户引导验证）
- **用户问题**：竞彩对心水方向降>2%时，是否有其他方向也降（分流筹码）？
- **逐场检查"降>2%"的场次后发现关键规律**：
  - 单出口降心水方向（只有心水方向在降）= **实盘信号，心水方向反而出了** → 0/2=0% → **不能排除**
  - 双出口降心水方向（其他方向也降）= **筹码分流** → 1/1=100%（样本极小）
- **完整信号体系**（前提：非友谊赛+心水>2.0）：
  - 升>3%（推离心水）→ 排除心水 → 4/4=100%
  - 降>2%+单出口 → **❌不能排除**（实盘信号，心水方向是真信号）
  - 降>2%+双出口 → 排除心水 → 1/1=100%
  - 变化<2%或不动 → 排除心水 → 7/10=70%
- **核心发现**：竞彩和心水方向一致（都推同一方向）时，心水是真信号不能排除；竞彩和心水方向矛盾（推离心水）时，赔率更诚实可以排除
