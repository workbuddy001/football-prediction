## Step 0 大小球方向判断 — 30场回测验证

### 数据基础
- 30场已录入大小球盘口+水位
- 27场有赛果可验证，19场有方向预测

### 整体准确率
- ✅ 9/19 = 47.4% (基础版)
- 实防6场: 5/6 = 83.3% ✅✅✅
- 假盘E 7场: 2/7 = 28.6% ❌
- 假盘F 6场: 2/6 = 33.3% ❌
- 模糊8场: 全部正确识别为不明确

### 核心发现（颠覆性！）

**发现1：极端0球 ≠ 陷阱，是真正的市场信号** ⚠️⚠️⚠️
- 框架原规则E："0球>>理论上限+5 → 假盘造热→实际小球"
- 实际验证：7场E触发中5场打出大球！
- 拜仁2-3(5球)、勒沃库森4-1(5球)、霍芬海姆3-3(6球)、阿贾克斯2-2(4球)、不来梅1-3(4球)
- **修正**：极端高开0球 = 真大球信号，不是陷阱！庄家看到的是真实的进球预期

**发现2：低水小球在近况大球背景下 = 真实小球** ⚠️
- 框架原规则F："近况大球+低水小球 → 低水方陷阱"
- 实际验证：这种模式的6场中4场打出小球
- 蔚山0-1(1球)、代格福什1-1(2球)、狼队1-1(2球)、AC奥卢2-0(2球)
- **修正**：线位在2.5-2.75 + 近况>5.0 + 低水小球 → 线位信号>近况信号，小球实防

**发现3：实防双信号共振 = 最可靠信号** ✅✅✅
- 任何两个信号同向（0球+线位 / 0球+水位 / 线位+水位）且无矛盾
- 6场实防中5场正确(83.3%)，唯一错误是信号内部存在矛盾被误判

### 修正后的规律总结

**真实大球信号（可靠）：**
1. 线位>标准线+0.5 + 大球低水 → 深线确认（勒沃库森4-1、比利亚雷亚尔5-1）
2. 0球大幅高开(>理论+2) + 线位偏深 + 大球低水（至少2个同向）
3. 近况>5.0 + 线位跟随加深(≥3.25) → 大球

**真实小球信号（可靠）：**
1. 线位<标准线-0.5 + 小球低水（≥2信号）→ 浅线确认（科莫0-0、仁川联0-1）
2. 0球<理论-2 + 线位偏浅 + 小球低水（三信号共振）
3. 线位2.25 + 小球低水 + 0球<10 → 小球强信号

**假盘/不确定：**
1. ⚠️ 极端高开0球(g0>理论+10) → 不再自动判定为陷阱，需结合线位判断
2. ⚠️ 线位2.5-2.75 + 低水0.7-0.85 → 低水方向可能是真实信号（需看0球方向验证）
3. 信号矛盾（0球⇌线位⇌水位三者方向不一致）→ 模糊
