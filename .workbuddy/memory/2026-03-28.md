## 筹码分流原则整合（3.28补充）

### 筹码分流核心原则（新增分析维度）

**原理：庄家通过调整三向赔率，将筹码引导均匀分布（分流），以控制赔付风险。**

| 筹码状态 | 判断条件 | 含义 | 操作建议 |
|----------|----------|------|----------|
| **全锁定** | 三向赔率均不动 | 庄家无引导，静观其变 | 纯按近况差/置信度 |
| **单向锁定** | 两向流动，一向锁定 | 锁定方向=庄家真实押注方向（强信号） | 按锁定方向下注 |
| **均衡分流** | 三向小幅变动（<2%） | 筹码被均衡引导，无强方向 | 按置信度方向 |
| **极端造热** | 某方向赔率大幅下降(>10%) | 筹码过度涌入，庄家不愿推该方向打出 | 反向信号强 |
| **极端推离** | 某方向赔率大幅上升(>10%) | 筹码被主动推离该方向 | 该方向实为冷门 |
| **单向造热** | 某方向赔率降5-10% | 中等造热 | 结合规律判断反向 |

### 筹码分流与置信度联动规则
- **锁定方向 = 置信度方向** → 稳定性+2（双重确认）
- **锁定方向 ≠ 置信度方向** → 爆冷警惕（庄家押注≠市场热门）
- **全锁+近况差≥8** → 主队超强信号
- **均衡分流+高置信(≥66%)** → 结果可信

1. **澳门心水=筹码放大器**：澳门推哪个方向，公众就跟哪边，直接加重赔付压力
   - 澳门推 + 造热同向 → 双重聚焦 → 反向信号极强（周五002验证）
   - 澳门推平 + 平赔不动 → 平局是真方向（周五012验证）
   - 澳门推平 + 平赔反升 → 平局被推离（周五010验证）

2. **赔率绝对值赔付压力规则**：
   - 赔率>3.5的方向 → 赔付压力高 → 庄家不愿打出（周五006客3.61→平局；周五008平3.85+客4.55→主胜）

3. **近况差决定赔率可动空间**：
   - 近况差≥8 → 赔率只能微动<2% → 微动本身=主队格局（规律O升级理解）

4. **三方向赔付压力排序**：三方向中赔率最低=赔付最小=打出概率最大

5. **三重聚焦=反向最强**：澳门推+造热+近况差同向 → 反向信号最强（周五002）
