## V3.3 核心方法论：比分反推验证

### 三要素
1. 赔率热度：历史命中率>15% = 过热排除
2. 主队能力：近5场主队是否打出过该进球数
3. 客队能力：近5场客队是否打出过该进球数

### 核心逻辑
排除法确定首选总进球数 → 列出可能比分 → 逐项验证三要素
→ 全部不成立 → 排除该总进球数 → 退到备选 → 重新推导
→ 直到找到可行比分 或 全军覆没→不投注
