## bug修复：排除信号区域漏显示
- **原因1**：`分歧警告` strength=0，被 `activeSignals.filter(s=>s.action)` + specialRules 双重过滤，且警告区域仅在 hasColdAlert||hasHeatTrap||hasCoverMode 时渲染，分歧单独出现时完全不显示
- **原因2**：`30家公司共识` action=''，被 action 过滤
- **修复**：新增 `hasConflict` 检测，分歧信号用全量 `signals` 渲染不受 action 过滤；30家共识单独渲染在底部
