### 用户核心洞察（2026-04-27）

#### 洞察1：大球区 vs 小球区的"赔率不变"含义不同
```
大球区（4-7球）赔率不变 = 正常！✅
  → 原因：赔率高 + 投注拉力少 + 庄家不需要调整

小球区（0-3球）赔率不变 = 异常！⚠️
  → 原因：赔率低 + 投注拉力多 + 庄家应该调整
  → 可能是"庄家故意不设防"（诱盘）
```

#### 洞察2：赔率跳跃诱导
```
现象（帕德博恩 VS 沙尔克04）：
  4球4.65 → 5球8.25（跳跃77%！）⚠️

含义：
  4球是"高诱导区" → 庄家诱导玩家买4球
  实际可能打出5球（或6球）✅
```

#### 洞察3：大球区"过度防范"陷阱
```
现象（巴黎FC VS 里尔）：
  近况5.6（极高）+ 6球↓23.5% + 7球↓32%

含义：
  庄家"过度防范"大球 = 可能是陷阱！⚠️
  → 诱导玩家买大球
  → 实际打出1球（小球）✅
```
