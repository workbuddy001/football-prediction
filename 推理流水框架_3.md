## 第五步：聚焦推荐

### 5.1 总进球数推荐

```
推荐逻辑：
如果 排除4-7球 + 排除0-1球
    → 剩下2球或3球
    → 看2球vs3球赔率：哪个更低？更低者优先

如果 排除4-7球 + 无法排除0-1球
    → 0-3球四选一
    → 需要更多信号（近况细节、伤病、战意等）
```

### 5.2 让球推荐

```
推荐逻辑：
如果 让胜<1.60（超低热）
    → 推荐让胜（57.1%置信度）

如果 让负<1.60（超低热）
    → 推荐让负（59.2%置信度）

如果 让胜>3.00（高赔）
    → 反向推荐让负（53.8%置信度）

如果 让负>3.00（高赔）
    → 反向推荐让胜（56.8%置信度）
```

### 5.3 两个进球数二选一的推理方法

当排除法后剩下两个进球数时，需要更细致的推理。以下是4种方法的优先级排序：

#### 方法1：比较赔率高低（最常用，优先级1）

**原理**：赔率越低 = 庄家越看好 = 打出概率越高

```
如果 赔率A < 赔率B
    → 庄家更看好A → 推荐A

如果 赔率A > 赔率B
    → 庄家更看好B → 推荐B
```

**示例**：
```
2球赔率 = 4.85
3球赔率 = 3.7

3球(3.7) < 2球(4.85) → 庄家更看好3球 → 推荐3球
```

---

#### 方法2：看隐含概率（最准确，优先级2）

**原理**：隐含概率 = 1 / 赔率，越高 = 庄家越看好

```
隐含概率 = 1 / 赔率

如果 隐含概率A > 隐含概率B
    → 庄家更看好A → 推荐A
```

**示例**：
```
2球隐含概率 = 1 / 4.85 = 20.6%
3球隐含概率 = 1 / 3.7 = 27.0%

3球隐含概率(27.0%) > 2球隐含概率(20.6%) → 推荐3球
```

**注意**：方法1和方法2通常结论一致，但方法2更准确（考虑了赔付平衡）。

---

#### 方法3：看赔率变化方向 + 幅度（优先级3）

**原理**：赔率下降 = 庄家防范；赔率上升 = 庄家不看好

```
如果 赔率A下降 + 赔率B上升
    → 庄家看好A，不看好B → 推荐A

如果 两个都上升 或 两个都下降
    → 看变化幅度：变化幅度小的 = 庄家相对更看好
```

**示例**：
```
2球变化：↑3.2%（上升 = 不看好）
3球变化：↑4.2%（上升 = 不看好）

两个都上升 → 看变化幅度：
2球升幅 = 3.2%
3球升幅 = 4.2%

2球升幅更小(3.2% < 4.2%) → 庄家相对更不排斥2球 → 推荐2球？
```

**⚠️ 警告**：当A和B的变化方向相同时（都升或都降），方法3的置信度较低，优先参考方法1和2。

---

#### 方法4：结合让球盘判断（辅助验证，优先级4）

**原理**：让球盘反映庄家对"胜负"的看法，可以辅助判断总进球数

```
让胜低赔 → 主队不败 → 总进球数偏向2-3球
让平低赔 → 小胜或平 → 总进球数偏向1-2球
让负低赔 → 客队大胜 → 总进球数偏向3+球
```

**示例**：
```
让球(+1)：让胜1.87 / 让平3.90 / 让负2.96

让胜1.87低赔 → 庄家看好"主队受让1球后不败"
= 实际比分：主队不败（0-0, 1-0, 1-1, 2-1, 2-0...）
= 总进球数：0, 1, 2, 3球

结合0球/1球赔率极高 → 排除0-1球
→ 剩下2-3球 ✓（和方法1、2一致）
```

---

#### 综合判断逻辑

```
Step 1: 用方法1（赔率高低）初步判断
Step 2: 用方法2（隐含概率）验证
Step 3: 如果方法1和2结论一致 → 直接推荐
        如果方法1和2结论不一致 → 用方法3和4辅助判断
Step 4: 如果4个方法都无法确定 → 双选（标注置信度）
```

---

#### 何时用双选？

```
如果 方法1和2推荐不一致
    → 双选（2-3球）

如果 方法1和2推荐一致，但方法3和4反向
    → 双选 或 小注

如果 4个方法都推荐一致
    → 单选（高置信度）
```

### 5.4 让平规律辅助判断

当让球盘出现"让平中赔规律"时，可以通过让球方向推导可能的总进球数范围。

#### 让平三个区间规律（来源：MEMORY.md 2026-04-25）

| 区间 | 前置条件 | 推荐 | 置信度 | 可能比分 | 总进球数范围 |
|------|---------|------|--------|---------|-------------|
| **中低(3.3~3.64)** | 客近况好(form_diff < -0.3) + 让胜赔更低 | 让胜 | 83%~81% | 1-1, 2-1 | 2-3球 |
| **中赔(3.65~3.95)** | 主受让 + 客近况好(form_diff < -0.3) + 让胜赔更低 | 让胜 | 88.2%~100% | 1-1, 2-1 | 2-3球 |
| **高区间(4.0~4.5)** | 主让球 + 客近况好(form_diff < -0.5) + 让负赔更低 | 让负 | 90% | 1-2, 1-3 | 3-4球 |

#### 条件2反向信号（2026-04-26 新增）

- **触发条件**：让平3.6-3.9（中赔区间）+ 客近况<2.5（客队近期进攻较差）
- **推荐**：让胜（历史命中率60.5%，23/38场）
- **辅助参考**：
  - 平均总进球2.35球 → 可考虑"小2.5球"
  - 主队零封概率60.9%
- **避免**：不买让平（打出率7.9%），谨慎买让负（打出率31.6%）

#### 推理步骤

```
Step 1: 检查让平赔率所在区间（3.3~3.64 / 3.65~3.95 / 4.0~4.5）
Step 2: 检查前置条件是否满足（客近况好？主受让？让胜/让负赔更低？）
Step 3: 根据规律推荐让球方向
Step 4: 根据让球方向列出可能比分
Step 5: 从可能比分中提取总进球数范围
Step 6: 结合赔率变化，在总进球数范围内进一步聚焦
```

### 5.5 庄家无变化（0次变化）的推理（2026-04-27第三次修正）

当某进球数的赔率**变化0次**时，需要分析庄家的真实意图。

**⚠️ 核心认知（2026-04-27用户洞察）**：
```
大球区（4-7球）赔率不变 = 正常！
  → 原因：赔率高 + 投注拉力少 + 庄家不需要调整
  → 不能因为"变化0次"就排除大球 ✅

小球区（0-3球）赔率不变 = 异常！需警惕 ⚠️
  → 原因：赔率低 + 投注拉力多 + 庄家应该调整
  → 可能是"庄家故意不设防"（诱盘）
```

#### 区分"大球区"和"小球区"

| 区域 | 赔率特点 | 投注拉力 | 变化0次含义 |
|------|---------|---------|-------------|
| **大球区（4-7球）** | 高赔率（>4.0） | 少 | **正常！** 庄家不需要调整 ✅ |
| **小球区（0-3球）** | 低赔率（<4.0） | 多 | **异常！** 可能是诱盘 ⚠️ |

#### 核心逻辑

```
如果庄家担心某个结果打出
    → 会下降赔率（减少赔付）
    → 或上升赔率（阻止投注）

如果庄家不担心某个结果打出
    → 不会调整赔率（变化0次）
    → "反正不会打出，随便你投"
```

**但！** 变化0次也可能是**庄家故意不设防**，制造"稳"的假象（诱盘）。

---

#### 情况1：低赔率 + 变化0次 = 可能是诱盘

**现象**：某进球数赔率最低（<4.0），但变化0次

**玩家心理**：
```
看到低赔率 → 以为"稳" → 大量投注
```

**庄家意图**：
```
如果低赔结果真的打出 → 庄家需要赔付大量筹码
    → 庄家应该"怕"（应该下降赔率减少赔付）

但庄家变化0次 → 不担心低赔结果打出
    → 说明低赔结果可能不打出
    → 低赔率是诱盘！
```

**判断**：低赔率（<4.0）+ 变化0次 → **可能是诱盘** → 该结果可能不打出 ❌

**案例（帕德博恩 VS 沙尔克04，2026-04-27）**：
```
近况6.2（极高）→ 理论预期4+球
静态赔率：2球3.55（最低）+ 3球3.5（次低）
变化次数：全部0次（庄家无变化）

分析：
  如果2-3球真的打出 → 庄家赔付大量筹码 → 应该下降赔率
  但庄家变化0次 → 不担心2-3球打出
  → 2-3球可能是诱盘 → 不打出 ❌

实际比分：2:3 = 5球 ✓（2-3球确实不打出）
```

---

#### 情况2：高赔率 + 变化0次 = 可能是真实不打出

**现象**：某进球数赔率最高（>6.0），但变化0次

**玩家心理**：
```
看到高赔率 → 以为"不可能打出" → 很少投注
```

**庄家意图**：
```
如果高赔结果打出 → 庄家需要赔付高倍数（6倍+）
    → 但投注量少 → 赔付总额不高 → 庄家风险可控

庄家变化0次 → 不担心高赔结果打出
    → 可能真的是"小概率事件"
```

**判断**：高赔率（>6.0）+ 变化0次 → **可能是真实不打出** → 该结果可能不打出 ❌

**注意**：如果高赔结果真的打出，庄家赔付倍数高但投注量少 → 风险可控。

---

#### 情况3：中赔率 + 变化0次 = 无法确定

**现象**：某进球数赔率中等（4.0-6.0），变化0次

**判断**：无法确定 → 需要结合**近况**、**让球盘**、**历史数据**等其他信号

---

#### 推理步骤

```
Step 1: 找出变化0次的进球数

Step 2: 对这些进球数分类：
   - 低赔率（<4.0）+ 变化0次 → 可能是诱盘 → 考虑排除
   - 高赔率（>6.0）+ 变化0次 → 可能是真实不打出 → 考虑排除
   - 中赔率（4.0-6.0）+ 变化0次 → 无法确定 → 需要其他信号

Step 3: 结合近况判断：
   - 如果近况极高（>4.0）+ 低赔率是2-3球 + 变化0次
     → 2-3球可能是诱盘 → 重点考虑4+球

Step 4: 结合让球盘判断：
   - 如果让球盘显示"大胜"预期 + 低赔率是2-3球 + 变化0次
     → 2-3球可能是诱盘 → 重点考虑4+球

Step 5: 如果排除低赔率进球数后，剩下高赔率进球数
   → 这就是庄家的真实防范方向 ✓
```

### 5.14 全场0次变化时的决策树（2026-04-27新增）

**⚠️ 核心发现（佐加顿斯VS哈马比复盘）**：全场0次变化时，**不能直接套用5.11节的大球推荐**，必须先判断庄家开盘意图。

#### 决策树

```python
# 全场所有进球数变化0次时的处理流程

Step 1: 计算0球vs理论差异
  理论0球 = 近况合计 + 10
  差异 = 实际0球 - 理论0球

Step 2: 检查让球盘方向
  让胜低赔（<2.0）+ 主受让 → 让球盘指向小球（1-3球）
  让负低赔（<2.0）+ 主让球 → 让球盘指向大球（3-7球）
  方向不明确 → 无法用此规则

Step 3: 判断

  条件A：0球显著低开（差异<-3）+ 让球盘指向小球
    → 庄家开盘看好小球 → 0次变化=坚定不动
    → 推荐小球（1-3球）
    → ❌ 不要套用5.11节的大球推荐！
    → 案例：佐加顿斯VS哈马比（0球14 vs 理论17.6，差-3.6，让胜1.60）→ 实际2球✅

  条件B：0球显著高开（差异>+3）+ 让球盘指向大球
    → 庄家开盘看好大球 → 0次变化=坚定不动
    → 推荐大球（4-7球）
    → ✅ 可以套用5.11节
    → 案例：帕德博恩VS沙尔克04（0球15 vs 理论16.2，差-1.2，但让负1.45指向大球）

  条件C：0球低开 + 让球盘指向大球（方向矛盾）
    → 信号冲突 → 0次变化=数据不足
    → 降低置信度，需要更多辅助信号判断

  条件D：0球高开 + 让球盘指向小球（方向矛盾）
    → 信号冲突 → 同上处理
```

#### 5.11节 vs 5.14节的区别

| 对比项 | 5.11节（原规则） | 5.14节（新规则） |
|--------|----------------|----------------|
| **触发条件** | 近况>5.0 + 全场0次变化 | 全场0次变化 + 检查0球vs理论 + 让球盘方向 |
| **结论** | 重点考虑大球 | **取决于庄家开盘意图**（可大球可小球） |
| **关键前置** | 无 | **必须先判断0球vs理论差异和让球盘方向** |
| **适用场景** | 庄家初始赔率完美反映近况 | **所有全场0次变化的比赛** |

#### 重要提醒

```
全场0次变化有两种截然不同的含义：

含义1：庄家"深思熟虑后不动" → 初始赔率准确 → 按初始赔率方向走
  → 判断方法：0球vs理论差异显著 + 让球盘方向一致

含义2：庄家"根本没管"（低关注度比赛） → 初始赔率不可靠 → 需要更多信号
  → 判断方法：0球vs理论差异不大 + 让球盘方向不明确

⚠️ 5.11节只覆盖了"含义1中的大球场景"，漏掉了"含义1中的小球场景"
⚠️ 5.14节是更通用的规则，优先级高于5.11节
```

### 5.15 全场同频波动 + 孤立不动 = 声东击西（2026-04-27新增）

**⚠️ 核心发现（曼城VS南安普敦复盘）**：当绝大多数方向同频波动时，变化数据是噪音（弱招摇），真实方向藏在"孤立不动"的那个方向。

#### 触发条件

```python
# 同时满足两个条件时触发

条件①：绝大多数方向（≥6个）变化次数相同（同频波动）
  → 例：7个方向都是3次变化

条件②：存在1个方向变化0次（孤立不动）
  → 例：3球0次变化，其他全部3次

# 附加验证（可选，提升置信度）：
条件③：孤立不动方向的赔率是最低或接近最低的
  → 例：3球3.60是全场最低赔率
```

#### 操作规则

```python
Step 1: 丢弃波动数据
  同频波动 = 噪音 = 弱招摇吸引投注到错误方向
  → 不要被"小球在降！大球在升！"牵着走

Step 2: 识别"孤立不动"方向
  → 唯一0次变化的方向 = 庄家真实意图所在

Step 3: 验证
  如果孤立不动方向的赔率也是最低 → 强信号 ✅
  → 推荐"孤立不动"方向

Step 4: 辅助确认
  让球盘方向应与孤立不动方向一致
  → 例：让(-2)让负2.10 → 总进球2-3球 → 与3球一致 ✅
```

#### 操盘手法还原

```
庄家操作模式：
  小球区(0-2球)：↓降3次 → "看我在压小球赔率！小球可能出哦！"
  大球区(4-7球)：↑升3次 → "看我在抬大球赔率！大球不会出哦！"
  3球：          0次不动 → "……"（沉默）

  → 7个方向同频波动 = 制造方向噪音（弱招摇）
  → 3球孤立不动 = 藏在噪音中不吸引注意
  → 赔率3.60最低 = 庄家真实看法就在这里，但不想你注意到

  本质：声东击西 — 用同频波动制造方向错觉，真实方向沉默不动
```

#### 案例：曼彻斯特城 VS 南安普敦（2039328）

```
变化次数：
  0球3次↓11.1%  1球3次↓6.2%  2球3次↓4.3%
  3球0次→（孤立不动，赔率3.60最低）
  4球3次↑1.2%  5球3次↑4.2%  6球3次↑4.8%  7球3次↑9.5%

Step 1: 7个方向同频3次变化 → 丢弃波动数据（噪音）
Step 2: 3球孤立不动 → 庄家真实意图
Step 3: 3球赔率3.60全场最低 → 强信号 ✅
Step 4: 让(-2)让负2.10(90%) → 总进球2-3球 → 一致 ✅

推荐：3球（优先），2球（备选）
实际比分：2-1 = 3球 ✅
```

#### 与5.14节的区别

| 对比项 | 5.14节（全场0次变化） | 5.15节（同频波动+孤立不动） |
|--------|---------------------|--------------------------|
| **触发条件** | 全场所有方向0次变化 | 绝大多数同频波动 + 1个孤立0次 |
| **0次变化含义** | 庄家"坚定不动" | 庄家"藏在噪音中" |
| **分析重点** | 0球vs理论 + 让球盘方向 | 丢弃波动数据 → 赔率结构 + 让球盘 |
| **操盘手法** | 深思熟虑后不动 | **声东击西** |

### 5.16 初赔比终盘更能反映庄家真实意图（2026-04-27新增）

**⚠️ 核心发现（挪威VS瑞士复盘）**：当近况极高+市场方向与庄家开盘相反时，**终盘是市场压力下的被动调整，初赔才是庄家的真实判断**。

#### 触发条件

```python
# 同时满足以下条件时触发

条件①：近况极高（>5.0）→ 市场大量买入大球
条件②：0球终盘≈理论（看起来"正常开盘"）
条件③：但0球初赔显著低开（初赔 < 理论-2）→ 庄家开盘认为小球
```

#### 操作规则

```python
Step 1: 计算0球初赔（从终盘和变化幅度反推）
  初赔 = 终盘 / (1 + 变化幅度)
  例：终盘17.0，变化+13.3% → 初赔 = 17.0 / 1.133 ≈ 15.0

Step 2: 初赔 vs 理论差异
  差异 = 初赔 - 理论0球
  如果差异 < -2 → 庄家初赔显著低开 → 庄家看好小球

Step 3: 判断终盘变化含义
  当近况极高时，市场会大量买入大球
  → 庄家被迫升高0球赔率（往大球方向调整）
  → 终盘≈理论 不是"庄家认可近况"
  → 而是"庄家被动跟随市场"

Step 4: 结论
  初赔低开（庄家真实意图）> 终盘≈理论（市场压力）
  → 推荐小球（庄家真实判断）
```

#### 为什么终盘会骗人

```
近况7.2极高 → 所有玩家都看好大球 → 买入大球
→ 庄家大球赔付风险增大 → 被迫调低大球赔率（升高0球赔率）
→ 终盘0球=17.0 ≈ 理论17.2 → 看起来"正常"
→ 但庄家的原始判断在初赔里（低开=看好小球）
→ 终盘只是市场压力下的结果，不是庄家改变了判断

⚠️ 第二步（理论vs实际）必须用初赔，不是终盘！
```

#### 案例：挪威 VS 瑞士（2038721）

```
近况：主4.0 + 客3.2 = 7.2（极高）
理论0球 = 17.2
终盘0球 = 17.0（↑13.3%，1次变化）

Step 1: 初赔 = 17.0 / 1.133 ≈ 15.0
Step 2: 初赔15.0 vs 理论17.2 → 差异-2.2（显著低开）→ 庄家看好小球
Step 3: 终盘17.0≈理论17.2 → 不是"认可近况"，是市场压力
Step 4: 推荐小球（0球或2球）

辅助信号：
  - 3球3次变化 = 弱招摇（注意事项11）→ 3球可疑
  - 让负1.71（客队受让后强）→ 总进球0-3球

推荐：0球（优先），2球（备选）
实际比分：0-0 = 0球 ✅
```

#### 与现有规则的关系

```
5.16节 vs 第二步（理论vs实际）：
  第二步当前用终盘比较 → 5.16节要求用初赔比较
  当近况极高时，初赔比终盘更准确

5.16节 vs 3球弱招摇：
  3球3次变化 = 弱招摇 → 可以排除3球
  但本场的关键信号是初赔低开，不是3球变化次数
```

### 5.17 "没必要做但做了" = 反向信号（2026-04-27新增）

**⚠️ 核心发现（黑山VS斯洛文尼亚复盘）**：当近况低+市场已经不看好某个方向时，庄家却主动去操作那个方向 → **这是多余的、故意的操作 → 可能是反向信号**。

#### 触发条件

```python
条件①：近况低（<3.0）→ 市场自然不看好大球
条件②：0球低开（庄家开盘也看好小球）
条件③：但庄家仍然主动降低高球(6-7球)水位 → 没必要做的事！

→ 三个条件同时满足时触发
```

#### 核心逻辑

```
正常情况：
  近况低 → 市场不看好大球 → 庄家不需要操作大球
  → 高球水位应该"自然"保持高位（没人买，不需要调）

反常情况：
  近况低 + 市场不看好大球 + 庄家却主动降低高球水位
  → 庄家在"多做"不必要的事
  → 没必要做的事 = 故意的

  为什么故意？
  → 可能是制造"大球更不会出"的安全感假象
  → 可能是庄家掌握赛前信息（如必须分出胜负）
  → 实际进球可能远超近况显示
```

#### 操作规则

```python
Step 1: 检查是否有"多余的"操作
  如果近况低 + 0球低开 → 市场和庄家都看好小球
  → 这时高球水位应该自然高位（不需要动）
  → 如果高球(5-7球)反而被主动降低 → 多余操作 ⚠️

Step 2: 多余操作的方向 = 反向信号
  庄家降低大球水位 → 不是"防范大球"
  → 而是"制造大球不出的安全感" → 反向
  → 实际可能出大球

Step 3: 降低置信度
  当检测到"没必要做但做了" → 原有小球信号全部降低置信度
  → 这场比赛庄家可能在"做局"，不能完全相信开盘信号
```

#### 案例：黑山 VS 斯洛文尼亚（2038722）

```
近况：主3.4 + 客1.0 = 4.4（客1.0极低）
0球：9.0（低开-5.4）
市场：不看好大球（近况低）
庄家操作：6球↓7.1%、7球↓9.5%（主动降低高球水位）

Step 1: 近况低 + 0球低开 → 不需要操作大球
Step 2: 庄家却降低了6-7球水位 → 多余操作 ⚠️
Step 3: 多余操作 → 反向信号 → 小球信号不可信

实际比分：2-3 = 5球（远超近况显示）✅
```

#### 与5.14节的关系

```
5.14节：全场不动 → 坚定不动 = 庄家有信心
5.17节补充：全场几乎不动 + 但有个别"多余操作" → 不是坚定不动，是做局

区分方法：
  全场完全不动（0次） → 5.14节适用
  几乎不动 + 有1-2个"多余操作" → 5.17节适用
  "多余"的判断标准：近况低+市场方向一致时，庄家没理由操作的方向
```

### 5.19 半全场赔率辅助判断全场方向（2026-04-28新增）

**⚠️ 核心发现（巴列卡诺3-3皇家社会复盘）**：当对全场胜平负方向举棋不定时，**半全场赔率变化**可以提供更强的方向信号。半全场拆出了3条通向同一结果的路径，如果3条路径同方向变化，信号强度远大于单一全场赔率变化。

#### 触发条件

```
条件①：对全场方向不确定（胜/平/负赔率接近，或信号矛盾）
条件②：存在半全场赔率变化数据（hafu_change）

→ 当两个条件同时满足时，触发5.19节辅助判断
```

#### 核心逻辑

```
全场"平"只有一个赔率 → 变化1次可能只是微调
半全场"平"拆出3条路径：
  平平（半场平→全场平）
  负平（半场客胜→全场平）
  胜平（半场主胜→全场平）

如果3条路径全部下降 → 庄家在所有"通向平局"的路径上都降赔 → 强烈防范平局！
信号强度：3条路径同方向 > 单一全场变化 3倍以上
```

#### 操作规则

```
Step 1: 识别全场方向信号
  → 胜/平/负哪个在下降？哪个在上升？

Step 2: 检查半全场通向该方向的3条路径
  → 以"平局"为例：平平、负平、胜平
  → 以"主胜"为例：胜胜、平胜、负胜
  → 以"客胜"为例：负负、平负、胜负

Step 3: 判断一致性
  如果 3条路径全部下降
    → 庄家对该方向有明确防范 → 强信号 ✅
    → 信号强度 = 全场变化的3倍

  如果 2条路径下降 + 1条上升
    → 庄家部分防范 → 中等信号

  如果 3条路径方向不一致（1升1降1不变）
    → 信号矛盾 → 不使用半全场辅助

Step 4: 与全场信号双重验证
  如果 全场变化 + 半全场3条路径全部同方向
    → 双重验证 → 方向确定性大幅提升 ⭐⭐⭐⭐⭐

  如果 全场变化 + 半全场路径矛盾
    → 需要更谨慎 → 降低全场信号的置信度

Step 5: 特殊信号——平平变化
  平平是3条路径中赔率最高（通常5-6+），庄家最不可能主动操作的路径
  → 如果平平也在下降 → 庄家连最不起眼的路径都在防范 → 最强信号！

  平平降>5% = 庄家最怕半场就平 → 可能是0-0或1-1的小球平
  负平降+胜平降 > 平平不变 = 庄家怕的是"追平" → 可能是2-2/3-3的大球平
```

#### 半全场辅助判断的适用场景

| 场景 | 是否适用 | 原因 |
|------|---------|------|
| 胜平负赔率接近，方向不明 | ✅ 强烈推荐 | 半全场提供额外维度 |
| 全场信号弱（变化<5%） | ✅ 推荐 | 3条路径可以放大弱信号 |

### 5.12 "赔率跳跃诱导"识别（2026-04-27新增）

**⚠️ 用户重大洞察**：当大球区出现"赔率跳跃"时，低赔率侧是**高诱导区**！

#### 什么是"赔率跳跃"？

```
正常情况（无诱导）：
  4球4.65 → 5球6.5 → 6球10.0 → 7球16.0
  → 赔率"平缓上升" → 无诱导 ✅

异常情况（有诱导）：
  4球4.65 → 5球8.25（跳跃！）→ 6球15.0（继续高）
  → 赔率"突然跳跃" = 庄家诱导信号 ⚠️
```

#### 为什么是"高诱导区"？

**玩家心理**：
```
1. 看到近况极高（>5.0）→ "肯定是大球！"
2. 看到4球4.65（相对低）→ "4球最有可能！"
3. 看到5球8.25（突然高）→ "5球不可能！"

→ 集中买4球
→ 实际打出5球（或6球）
→ 庄家通吃！✅
```

#### 识别方法

```
如果 近况合计 > 4.0（偏高或极高）
  + 4球赔率 < 5.0
  + 5球赔率 > 8.0（突然跳跃！）
  
  → 4球是"高诱导区" ⚠️
  → 不要买4球！
  → 重点考虑5球或更
```

**量化标准**（待验证）：
```
跳跃幅度 = (5球赔率 - 4球赔率) / 4球赔率
如果 跳跃幅度 > 50%（如4.65→8.25，跳跃77%）
  → 强诱导信号！⚠️
```

```

  7球：23.0

  → 集中买4球（因为4.65相对低）
  → 避开5球（因为8.25突然高）

```

#### 修正后的规则

```

  + 大球区出现"赔率跳跃"（4球<5.0，但5球>8.0）

#### 5.23.2 新陷阱模式：H模式——线位偏浅+低水造热
