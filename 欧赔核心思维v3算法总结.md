# 基于《欧赔核心思维》的算法优化总结

## 核心方法提取

从《欧赔核心思维》提取的关键方法：

### 1. 实盘 vs 韬盘（诱盘）
- **实盘**：降低赔付思维，信心强的开盘
- **韬盘**：降低信心思维，诱盘意图
- **中庸盘**：介于两者之间

### 2. 平赔分散作用
- **拉低平分散胜赔**：平赔下降但不过分，分散胜赔投注
- **抬胜分平**：抬高血压分散平局
- 平赔是最有效的分散工具

### 3. 判断逻辑
- 状态差距大 + 低赔 = 实盘（可信）
- 状态相近 + 平赔小幅下降 = 实盘
- 状态相近 + 平赔大幅下降 = 诱盘（防冷）
- 澳门推荐平局 = 关注平局
