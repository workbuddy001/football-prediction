# 3.28 全场造热比赛详细分析列表

> 造热定义：赔率变化总和>10%，或单方向变化>5%（造热/推离），或竞彩vs澳门出现异常对比

---

## 一、造热级别分类

| 级别 | 标准 | 场次数 |
|:----:|------|:------:|
| 🔴🔴 **极端造热** | 澳门+竞彩同时造热同方向，总和>30% | 1场 |
| 🔴 **大幅调控** | 澳门变化总和>15%，有明确方向性 | 1场 |
| 🟠 **中等造热** | 单家变化总和10-15%，或单方向>5% | 3场 |
| 🟡 **微弱造热** | 单方向变化3-5%，或竞彩造热但澳门不动 | 3场 |
| ⚪ **无造热** | 三向变化均<2%或总和<10% | 7场 |

## 七、造热规律总结

### 本期造热的3大模式

| 模式 | 场次 | 特征 | 操作 |
|------|:----:|------|------|
| **A. 庄家真造热** | 001 | 澳门+竞彩同向造热，无分流出口 | **反向** |
| **B. 庄家推离** | 005/002 | 澳门主动推离某方向，竞彩反向 | 规律五/观望 |
| **C. 市场造热+庄家不跟** | 009/007/006/003/008 | 竞彩造热但澳门不动 | 看盘口硬度信号 |

### 模式C的关键判断

```
竞彩造热 + 澳门不动：
```
