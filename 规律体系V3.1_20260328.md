## 核心原则：筹码分流+澳门心水联动

**总原理**：庄家通过调整三向赔率，将公众筹码引导均匀分布以控制赔付风险。**澳门心水是筹码聚焦的放大器**，赔率绝对值决定赔付上限，两者必须联动分析。

## V3.1核心修正总结

### 修正1：规律S — 增加澳门推方向≠造热方向条件

| 条件 | V3 | V3.1 |
|------|------|------|
| 近况差 | \|近况差\|≤2 | \|近况差\|≤2 |
| 赔率变化 | >5% | >5% |
| 澳门方向 | 无限制 | **≠造热方向** 且 **≠平局** |
| 赔率绝对值 | 无限制 | **反向方向<3.4** |

**口诀**：近况持平+赔率大动+澳门不推那个方向+反向方向赔率不太高→反向

### 修正2：规律五V3→V3.1 — 增加澳门推平条件

| 条件 | V3 | V3.1 |
|------|------|------|
| 主胜升幅 | >5% | >5% |
| 客胜变化 | >-5% / <-5% | >-5% / <-5% |
| 近况差 | \|差\|>3 | \|差\|>3 |
| 澳门方向 | 无限制 | **推平局** |

**逻辑**：澳门推主时，主胜升幅>5%=市场正常调整（不是推离造热），规律五不触发

### 修正3：规律U/V — 增加澳门推方向≠造热方向条件

| 条件 | V3 | V3.1 |
|------|------|------|
| 近况差 | ≥+8 / ≤-8 | ≥+8 / ≤-8 |
| 造热方向 | 降>5% | 降>5% |
| 澳门方向 | 无限制 | **≠造热方向** |

**逻辑**：澳门也推造热方向=实盘确认（不是诱导），规律U/V不触发

### 修正4：规律N — 增加客胜赔率绝对值排除

| 条件 | V3 | V3.1 |
|------|------|------|
| 主升 | >5% | >5% |
| 澳门 | 推客 | 推客 |
| 客降 | <-10% | <-10% |
| 客赔绝对值 | 无限制 | **≥2.5** |

**逻辑**：客赔<2.5=庄家能承受赔付=实盘确认，规律N不反向

### 修正5：规律J — 增加平赔不降条件

| 条件 | V3 | V3.1 |
|------|------|------|
| 澳门 | 推平 | 推平 |
| 平赔 | <3.0 | <3.0 |
| 赔率变化 | 主升+客降 | 主升+客降 |
| 平赔变化 | 无限制 | **≥-2%（不大幅下降）** |

**逻辑**：平赔也大幅下降=平局在承接筹码，客胜不是唯一方向

## V3.1完整规律体系

### A. 筹码状态分类

| 筹码状态 | 判断条件 | 含义 |
|----------|----------|------|
| 全锁定 | 三向赔率均不动(<0.5%) | 庄家静观，无引导信号 |
| 单向锁定 | 两向变化>2%，一向锁定<0.5% | 被锁定方向=庄家真实押注 |
| 均衡分流 | 三向均变化0.5-2% | 筹码被均衡引导 |
| 极端造热 | 某方向赔率变>10% | 筹码过度涌入 |
| 单向造热 | 某方向赔率变5-10% | 中等造热 |
| 正常波动 | 变化<2% | 正常市场调整 |

### B. 澳门心水联动规则

| 澳门推荐 vs 赔率变化方向 | 筹码聚焦程度 | 判断结论 |
|------------------------|------------|---------|
| 澳门推=赔率造热（同向） | 双重聚焦 | ⚠️ 赔付压力爆表，反向信号极强 |
| 澳门推≠赔率变化（反向） | 信号对冲 | 看赔率绝对值和近况差决胜 |
| 澳门推平+平赔微降/不变 | 平局筹码聚焦 | 平局是真方向 |
| 澳门推平+平赔反升 | 平局被主动推离 | 平局不打出 |

### C. 赔率绝对值赔付压力规则

| 赔率绝对值 | 赔付压力 | 打出概率 |
|-----------|---------|---------|
| < 2.5 | 低 | 高（庄家可承受） |
| 2.5-3.4 | 中 | 中 |
| > 3.4 | 高 | 低（庄家赔付压力大） |

### D. 近况差与赔率可动空间规则

| 近况差 | 赔率可动空间 | 逻辑 |
|-------|------------|------|
| ≥ +8 | 极小（<2%）| 主队热度天然高 |
| +4~+7 | 中（2-5%）| 适度引导分流 |
| -3~+3 | 大 | 双方均衡 |
| ≤ -8 | 极小（<2%）| 客队热度高 |

### E. 规律S（V3.1核心修正）
```
|近况差|≤2 + 赔率变化>5% + 澳门推方向≠造热方向 + 澳门≠平局 + 反向方向赔率<3.4
→ 反向
```

### F. 规律五V3.1
```
主升>5% + |近况差|>3 + 澳门推平 → 平局
主升>5% + 客降>5% + |近况差|>3 + 澳门推平 → 客胜
```

### G. 规律U/V（V3.1修正）
```
近况差≥+8 + 主胜降幅>5% + 客胜升幅>5% + 澳门≠主胜 → 防平
近况差≤-8 + 客胜降幅>5% + 主胜升幅>5% + 澳门≠客胜 → 防平
```

### H. 规律N（V3.1修正）
```
主升>5% + 澳门推客 + 客降>10% + 客赔≥2.5 → 反向主胜
```

### I. 规律J（V3.1修正）
```
澳门推平 + 平赔<3.0 + 主升 + 客降 + 平赔变化≥-2% → 客胜
```
