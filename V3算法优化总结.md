# V3算法优化总结

## 优化内容（基于《欧赔核心思维》PDF）

### 1. 吸收的核心思维

| 核心概念 | 应用 |
|---------|------|
| 人气指数体系 | 区分一档/准一档/二档球队 |
| 即时广义实力差 | 综合考虑状态、战意、往绩等 |
| 平赔分散思维 | 拉低平分散胜赔、抬胜分平 |
| 真盘思维 | 初盘即终盘=强烈信号 |
| 诱导分散 | 反向操作=诱导分散 |
| 极限诱导 | 超低赔+低平=陷阱盘 |

### 2. 测试结果对比

| 版本 | 3.14(冷门日) | 3.15(正路日) | 3.13 |
|------|-------------|-------------|------|
| V3原版 | **81.2%** | 48.3% | 58.3% |
| V3优化版 | 50.0% | 待测 | 50.0% |

### 3. 关键发现

1. **V3原版在冷门日表现优异(81.2%)**
   - 原因：原版有完整的实盘/诱盘判断
   - 诱盘识别是冷门日的关键

2. **V3优化版平局判断过于激进**
   - 增加了防平判断，但效果不佳
   - 导致该直接预测的比赛被判断为防平

3. **A级信号100%准确**
   - 极低赔(<1.35)+状态差距大 = 强队胜
   - 这类比赛可以重注

### 4. 最终建议

**继续使用V3原版算法**，原因：
- 3.14冷门日达到81.2%
- 3.15正路日有V4补充(96.6%)
- 算法组合可以覆盖不同比赛日

**V3优化版保留有价值部分：**
- A级信号100%准确，可作为高置信度信号
- 人气指数体系可辅助判断球队定位
